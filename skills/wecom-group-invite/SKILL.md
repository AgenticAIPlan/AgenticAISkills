# 企业微信口令入群 Skill

## 适用场景

当社群运营人员收到用户在企业微信聊天窗口发送的入群口令（如 "OCR"、"LLM"、"RAG"）后，需要快速查找对应技术交流群的邀请链接或活码，并生成标准回复话术时，使用本 Skill。

## 输入要求

- 用户发送的入群口令关键词（支持文本输入或从截图 OCR 提取）
- 口令与群的映射表已维护在 `references/groups.json` 中

## 执行步骤

1. 从用户输入中提取入群口令关键词。若输入为截图，先用系统 OCR 识别文字后提取口令。
2. 对口令做归一化处理：去除首尾空白，统一转为大写。
3. 读取 `references/groups.json`，用归一化后的口令作为 key 查表。
4. 若命中，按「输出要求」格式返回群信息与回复话术。
5. 若未命中，提示未找到该口令，并列出所有可用口令供运营核对。

## 输出要求

- 输出包含三个字段：群名称、邀请方式（链接或活码图片路径）、建议回复话术
- 格式如下：

  ```
  群名称：{name}
  邀请方式：{invite}
  建议回复：
  ---
  {reply_template}
  ---
  ```

- 若邀请方式为本地活码图片路径，需额外提示运营从该路径取图上传
- 不发起任何网络请求，不调用企业微信 API
- 不自动修改 `references/groups.json`

## 参考资料

口令与群的映射表维护在 `references/groups.json`，新增或下线群时请手动编辑该文件并提交 PR。
