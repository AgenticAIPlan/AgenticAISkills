# Style Variants

## Apple Blueprint

Use when the user wants:

- proposal-grade polish
- whitepaper credibility
- high-end restraint

Characteristics:

- sparse labels
- balanced whitespace
- elegant title hierarchy
- minimal connectors

## Palantir Ops

Use when the user wants:

- analytical clarity
- operator-facing systems feel
- stronger matrix structure

Characteristics:

- denser rows
- more explicit labels
- strong information grouping
- less decorative softness

## Mission Control

Use when the user wants:

- field operations feeling
- stronger directional flow
- command-center energy

Characteristics:

- stronger rails and directional movement
- more assertive grouping
- more tactical rhythm
- still restrained enough for formal documents
