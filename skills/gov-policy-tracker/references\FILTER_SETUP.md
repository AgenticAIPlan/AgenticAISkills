# 飞书多维表格筛选配置指南

## 一、创建筛选视图

### 1. 人工智能政策视图
1. 打开飞书多维表格：https://fcnhy42ew3lq.feishu.cn/base/J9Jpb3k23a68gqsoXPXclzyznMe
2. 点击左下角"+ 新建视图"
3. 选择"表格"
4. 命名为"人工智能政策"
5. 点击"筛选"按钮
6. 添加筛选条件：
   - 字段：关键词标签
   - 条件：包含
   - 值：人工智能
7. 保存视图

### 2. 数据要素政策视图
1. 点击"+ 新建视图"
2. 选择"表格"
3. 命名为"数据要素政策"
4. 点击"筛选"按钮
5. 添加筛选条件：
   - 字段：关键词标签
   - 条件：包含
   - 值：数据要素
6. 保存视图

### 3. 地方政策视图（重点地区）
1. 点击"+ 新建视图"
2. 命名为"地方重点政策"
3. 添加筛选条件：
   - 字段：地区
   - 条件：是以下任意一个
   - 值：北京、上海、广东、深圳、浙江、江苏、湖北、四川、福建、厦门、重庆
4. 保存视图

## 二、设置链接可点击

### 方法1：使用超链接字段（推荐）
1. 在表格中新建字段"原文链接（可点击）"
2. 字段类型选择"超链接"
3. 使用公式字段自动转换：
   ```
   HYPERLINK(原文链接, "点击查看")
   ```

### 方法2：直接在文本字段中点击
- 飞书多维表格的文本字段中的URL会自动识别为可点击链接
- 确保URL格式完整（以http://或https://开头）

## 四、数据展示优化

### 1. 确保无空白
系统已自动处理：
- 所有字段都有默认值
- 空值会显示为"未填写"或"无"
- 摘要自动生成（如果原始数据不足）

### 2. 来源部门全名
系统自动转换：
- 工信部 → 中华人民共和国工业和信息化部
- 北京市 → 北京市人民政府
- 深圳市 → 深圳市人民政府
- 等等

### 3. 政策摘要详细
系统会自动生成详细摘要，包括：
- 政策类型描述
- 关键词相关内容
- 标题核心信息
