## 六、数据字段说明

| 字段 | 说明 |
|------|------|
| 抓取日期 | 系统自动填充当前时间 |
| 政策发布日期 | 从网页解析或默认当前时间 |
| 来源部门 | 完整的政府部门名称 |
| 地区 | 自动识别（全国或地方） |
| 政策标题 | 完整政策标题 |
| 政策摘要 | 自动生成详细描述 |
| 原文链接 | 完整可点击的URL |
| 文件原名 | 从URL解析的文件名 |
| 政策类型 | 通知/意见/办法/规定等 |
| 关键词标签 | 人工智能、数据要素 |
| 优先级 | 高/中/低 |
| 处理状态 | 待处理/处理中/已完成 |
| 相关政策 | 关联的其他政策 |
| 备注 | 抓取时间等信息 |
