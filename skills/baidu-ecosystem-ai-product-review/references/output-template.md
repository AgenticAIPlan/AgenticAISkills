# 输出模板

```text
审核结论：通过 / 不通过 / 需澄清

不通过问题：
1. 位置：<字段名 / 页码 / 截图位置>
   问题：<为什么不通过>
   修改建议：<如何修改>

告警：
1. 位置：<字段名 / 页码 / 截图位置>
   问题：<需要人工关注的风险>
   建议：<如何补充或复核>

需提报人澄清：
1. 位置：<字段名 / 页码 / 截图位置>
   问题：<当前无法确认的点>
   建议补充内容：<需要补什么>
```

## 要求

- 位置尽量具体
- 问题描述要对应审核口径
- 修改建议必须可执行
- 对 OCR 低质量页要明确写“建议补更清晰扫描件”
