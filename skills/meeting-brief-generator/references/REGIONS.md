## 十一、地区使用指南

### 地区判断逻辑

```
1. 根据会谈地点/对象 → 判断地区维度
2. 查询该地区负责人 → 确认对接人
3. 查询该地区合作基础 → 了解已有合作
4. 查询该地区业务特色 → 确定谈参重点
5. 查询该地区政策环境 → 了解政策支持
```

### 地区谈参重点

| 地区类型 | 谈参重点 |
|----------|----------|
| 中心城市（深圳/成都/厦门） | 中心运营、生态建设、产业落地 |
| 政府合作（长沙/南昌） | 政策支持、国资合作、示范项目 |
| 产业特色（武汉/东莞） | 产业应用、案例打造、企业服务 |
| 国际窗口（香港） | 国际合作、数据引入、品牌出海 |
