# 对象类型模板库

不同会谈对象需要不同的谈参结构和表达风格。

---

## 模板一：企业客户/伙伴会谈

### 适用场景

- 商业公司
- 技术合作伙伴
- 生态伙伴（Powered by伙伴）
- AI创业团队

### 重点写什么

| 模块 | 重点内容 |
|------|----------|
| 对方画像 | 创始人/CEO背景、技术实力、产品能力、融资情况、行业地位、商业化进展 |
| 关键人 | 教育/职业背景、代表成果、行业影响力、获奖情况 |
| 过往合作 | 技术集成、产品共创、落地案例、生态活动、商业化进展 |
| 未来规划 | 产品路线图、市场拓展、合作深化 |

### 弱化什么

- 过于详细的组织架构
- 内部管理细节
- 非业务相关的社会活动

### 话术风格

强势、国际化、产业级，强调商业价值和技术领先性。

### 典型表达示例

**对方认可：**
> [对方名称]在[领域]拥有深厚的技术积累和行业领先的产品能力，[创始人/CEO]博士在[领域]有着超过[X]年的全球领军经验，团队已获得[投资机构]的融资支持，在[市场]具备强劲的增长潜力。

**合作肯定：**
> 过去一年，双方从技术适配、产品共创到生态推广已经形成了非常好的合作基础。特别是在[具体产品]的深度适配，以及[联合产品]方面，已经具备了进一步规模化推广的条件。

## 模板二：高校/科研机构会谈

### 适用场景

- 国家重点实验室
- 高校科研团队
- 研究院所
- 产学研合作基地

### 重点写什么

| 模块 | 重点内容 |
|------|----------|
| 对方画像 | 机构定位（国家级/省级）、研究方向、国家任务、数据资源、科研成果 |
| 关键人 | 学术背景、代表成果（顶刊论文、重大奖项）、行业影响力、社会贡献 |
| 过往合作 | MoU签署、项目合作、数据引入、联合研究 |
| 未来规划 | 垂类大模型、数据合作、产学研转化、国际交流 |

### 弱化什么

- 过于商业化的表达
- 盈利模式讨论
- 短期业绩要求

### 话术风格

稳重、学术性、国家使命感，强调科研价值和社会意义。

### 典型表达示例

**对方认可：**
> [机构名称]作为国家重点实验室，在[研究领域]处于国际领先水平。[教授姓名]团队在[具体领域]有着深厚的科研积累，多次在《Nature》《Science》等顶级期刊发表突破性成果，为[国家战略]做出了重要贡献。

**合作肯定：**
> 双方基于[合作备忘录/MoU]已经建立了良好的合作基础。此次交流将进一步推动[具体方向]的产学研协同，实现科研成果与产业应用的深度融合。

## 模板三：政府/协会会谈

### 适用场景

- 政府部门
- 行业协会
- 产业联盟
- 示范园区

### 重点写什么

| 模块 | 重点内容 |
|------|----------|
| 对方画像 | 机构职能、服务范围、辖区资源、政策优势 |
| 关键人 | 职务、分管领域、工作重点 |
| 过往合作 | 政策支持、项目对接、平台建设 |
| 未来规划 | 产业带动、示范效应、区域合作 |

### 弱化什么

- 政治敏感话题
- 利益分配细节
- 未公开政策

### 话术风格

庄重、战略性、服务导向，强调产业贡献和社会价值。

### 典型表达示例

**对方认可：**
> [机构名称]在[领域]发挥着重要的引导和推动作用，在[具体政策/规划]的指导下，为[产业发展]提供了有力支撑。

**合作肯定：**
> 双方在[具体项目]的合作取得了积极成效，下一步我们将继续深化合作，共同推动[领域]的创新发展。

## 模板四：开源社区/国际组织会谈

### 适用场景

- 开源项目/社区
- 国际标准组织
- 开发者联盟
- 国际技术社区

### 重点写什么

| 模块 | 重点内容 |
|------|----------|
| 对方画像 | 社区规模、项目活跃度、全球影响力、技术贡献 |
| 关键人 | 社区角色、技术背景、开源贡献 |
| 过往合作 | 代码贡献、技术集成、社区活动 |
| 未来规划 | 联合开源、开发者生态、国际推广 |

### 弱化什么

- 商业化压力
- 闭源技术细节
- 市场竞争

### 话术风格

开放、技术导向、社区感，强调协作和共享。

### 典型表达示例

**对方认可：**
> [项目/组织名称]作为全球领先的[领域]开源项目，拥有广泛的开发者社区和活跃的技术生态，在[技术方向]做出了重要贡献。

**合作肯定：**
> 双方在[具体合作]方面建立了良好的协作关系，下一步我们将继续深化开源合作，共同推动[技术方向]的发展。

## 模板五：签约仪式会谈

### 适用场景

- 正式签约仪式
- 战略合作协议签署
- MoU签署
- 重大合作启动

### 重点写什么

| 模块 | 重点内容 |
|------|----------|
| 合作意义 | 为什么签约、对双方的价值、行业影响 |
| 合作内容 | 具体合作范围、重点项目、预期成果 |
| 合作目标 | 短期目标、中长期愿景 |
| 后续计划 | 谁来落实、什么时间节点、什么形式 |

### 弱化什么

- 争议点、未尽事项
- 未达成的条件
- 过程中的困难

### 话术风格

正式、庄重、充满希望，强调历史意义和未来愿景。

### 典型表达示例

**签约意义：**
> 今天的签约标志着双方合作迈上了新的台阶。基于双方在[各自优势]的深度互补，我们将共同推动[领域]的创新与发展。

**合作展望：**
> 我们期待以此次签约为起点，在[具体方向]开展深度合作，实现互利共赢、共同发展。

## 会谈目标分类与策略

| 会谈目标 | 谈参重点 | 预期成果 |
|----------|----------|----------|
| 关系建立 | 对方背景、共同价值、合作可能性 | 建立初步联系、探索合作意向 |
| 合作推进 | 已有合作、进展、下一步动作 | 明确合作路径、落实具体项目 |
| 项目落地 | 项目方案、资源投入、时间计划 | 签署项目协议、启动实施 |
| 签约 | 合作意义、内容、目标、计划 | 正式签署合作协议 |
| 资源争取 | 我方价值、对方需求、合作模式 | 获得数据、资金、渠道等资源 |

## 地区维度模板

### 模板六：中心城市会谈（深圳/成都/厦门）

### 适用场景

- 人工智能创新中心已落地的城市
- 中心运营、生态建设、产业落地

### 重点写什么

| 模块 | 重点内容 |
|------|----------|
| 地区背景 | 中心建设进展、政策环境、产业基础 |
| 已有合作 | 中心运营成果、生态伙伴、标杆案例 |
| 业务特色 | 该地区主导产业、特色应用场景 |
| 政府关系 | 关键部门、政策支持、合作历史 |

### 话术示例

> 我们在[地区]已经建立了人工智能创新中心，与[合作方]开展了深入合作。在[主导产业]领域，我们已经打造了[具体案例]，期待进一步深化合作，共同推动[领域]的创新发展。

### 典型案例参考

- **深圳**：中心建设、OPC社区、高新投合作
- **成都**：高新区合作、赛力斯汽车AI、川草防火预警
- **厦门**：人工智能创新中心、海丝法研、农业基地

---

### 模板七：政府合作会谈（长沙/南昌）

### 适用场景

- 与地方政府、国资平台合作
- 政策支持、国资合作、示范项目

### 重点写什么

| 模块 | 重点内容 |
|------|----------|
| 地区背景 | 政策环境、国资平台、产业规划 |
| 政府关系 | 关键领导、合作历史、政策支持 |
| 已有合作 | 国资合作项目、示范应用、中心建设 |
| 未来规划 | 产业带动、标杆打造、区域合作 |

### 话术示例

> 我们高度重视与[地区]的合作，在[政策/规划]的支持下，我们已与[国资平台/政府]开展了[具体项目]。下一步，我们期待共同打造[标杆项目]，为[地区]的[产业发展]贡献力量。

### 典型案例参考

- **长沙**：长沙国资、英维智汇、国资平台合作
- **南昌**：南昌中心建设、OPC社区规划

---

### 模板八：产业特色会谈（武汉/东莞）

### 适用场景

- 有明确产业特色和应用场景的地区
- 工业应用、制造业升级、智慧城市

### 重点写什么

| 模块 | 重点内容 |
|------|----------|
| 地区背景 | 产业结构、主导产业、产业政策 |
| 产业应用 | 具体应用场景、技术需求、合作案例 |
| 已有合作 | 企业合作、应用落地、技术集成 |
| 未来规划 | 产业升级、案例复制、生态建设 |

### 话术示例

> 我们在[地区]聚焦于[主导产业]的智能化升级，与[企业]在[具体场景]开展了合作。基于[技术能力]，我们帮助[企业]实现了[具体成果]，期待进一步扩大合作，共同推动[产业]的数字化转型。

### 典型案例参考

- **武汉**：OPC社区、象点装配视觉检测、艾普工华MES/MOM
- **东莞**：制造业升级、滨海湾新区产业合作

### 模板九：国际窗口会谈（香港）

### 适用场景

- 国际合作窗口
- 数据引入、品牌出海、国际合作

### 重点写什么

| 模块 | 重点内容 |
|------|----------|
| 地区背景 | 国际化优势、政策环境、战略定位 |
| 国际合作 | 高校合作、数据引入、国际平台 |
| 已有合作 | CUHK实验室、香港会客厅、京港合作 |
| 未来规划 | 国际市场拓展、数据生态中心、品牌出海 |

### 话术示例

> 香港是我们拓展国际市场的重要窗口。我们与港中文等高校建立了深度合作关系，共同推进[领域]的产学研融合。下一步，我们期待基于香港的平台优势，共同构建国际AI合作网络。

### 典型案例参考

- **香港**：CUHK农业生物实验室、香港会客厅、京港合作、新加坡合作规划
