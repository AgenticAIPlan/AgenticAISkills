# 公开信息检索与整理风格参考

本文档定义系统自动检索公开信息时的检索范围、整理风格和输出格式。

---

## 一、公开信息检索规则

### 检索触发条件

当会谈对象涉及以下情况时，系统自动检索公开信息：

| 对象类型 | 检索条件 | 检索内容 |
|----------|----------|----------|
| 政府官员 | 姓名明确、职务明确 | 职务、履历、工作重点、近期动向 |
| 知名企业 | 企业名称明确 | 企业简介、行业地位、核心业务、近期动态 |
| 代表性人物 | 姓名明确、有影响力 | 教育/职业背景、代表成果、行业影响力 |
| 高校/科研机构 | 机构名称明确 | 机构定位、研究方向、科研成果 |

### 检索范围

- **内部知识库**：优先查询 REGIONS.md、CASES.md 中已有信息
- **用户提供的素材**：匹配相关人物、企业、政策信息
- **系统默认**：对于检索不到的信息，标注"待检索"或"待补充"

## 二、人物信息整理风格

### 参考格式（基于厦门、香港素材）

```
【姓名】[姓名]
【职务】[当前职务]

【简介】
[个人简介，1-2句话]

【履职信息】
- 职务1：时间
- 职务2：时间
- 职务3：时间

【人物经历】
- 时间：单位：职务
- 时间：单位：职务
- 时间：单位：职务

【最新动向】
- 时间：事件
- 时间：事件
- 时间：事件

【重点关注领域】
- 领域1
- 领域2
- 领域3
```

### 整理要点

1. **信息层级**：从基本信息 → 履职信息 → 人物经历 → 最新动向 → 重点关注领域
2. **时间线方式**：用"时间：事件"格式罗列
3. **内容控制**：每个部分控制在3-5项
4. **语言风格**：客观、简洁、基于事实
5. **可引用表达**：提炼1-2句适合领导提及的尊重性表达

### 示例（陈通汕）

```
【姓名】陈通汕
【职务】厦门市思明区委副书记、区长

【简介】
厦门市思明区主要负责人，推动区域经济发展和数字化转型。

【履职信息】
- 2025年：任厦门市思明区委副书记、区长
- [之前职务]

【人物经历】
- [过往经历]

【最新动向】
- 2025年7月：主持中共思明区委十二届九次全会
- 2025年5月：调研鼓浪屿小型消防站建设
- 2025年4月：参与厦门人工智能产业"四链融合"项目对接会
- 2025年4月：调研数字立方大厦，推动AI产业集群建设
- 2025年4月：调研中华五育馆建设
- 2025年2月：到访中智集团座谈交流
- 2025年2月：到发展集团调研交流

【重点关注领域】
- 数字经济发展
- AI产业集群建设
- 城市现代化治理
- 区域协调发展
```

## 三、企业信息整理风格

### 参考格式（基于厦门、香港素材）

```
【企业名称】[企业名称]
【地区】[地区]
【负责人】[负责人]

【合作背景】
[对方简介、合作起点]

【合作成熟度】
[阶段1-5]

【合作内容】
- [具体合作1]
- [具体合作2]
- [具体合作3]

【可对外表达亮点】
[可用于PR的表达，引用具体数据]

【可供领导引用话术】
[可直接引用的表达]

【后续推进】
[下一步计划]
```

### 整理要点

1. **合作成熟度**：明确当前阶段（1-5）
2. **合作内容**：用列表形式，每项1句话
3. **可对外表达亮点**：突出数据、成果、影响力
4. **话术引用**：1-2句可直接引用的表达
5. **后续推进**：明确下一步计划

### 示例（新能安）

```
【企业名称】新能安
【地区】厦门
【负责人】

【合作背景】
新能源头部企业，智能制造技术攻坚

【合作成熟度】
阶段3（落地案例）

【合作内容】
- 电池外观质检方案深化
- 关键工序检测覆盖率达95%
- 整体提升约70%
- 部署达800余套

【可对外表达亮点】
帮助新能安实现电池外观质检智能化，关键工序检测覆盖率达95%，整体提升约70%，部署达800余套，显著提升生产效率和品质控制。

【可供领导引用话术】
我们与新能安在电池外观质检方面的合作取得了显著成效，关键工序检测覆盖率达95%，整体提升约70%，部署达800余套，为新能源行业的智能化升级提供了有力支撑。

【后续推进】
深化AI质检能力，探索更多应用场景
```

## 四、政策信息整理风格

### 参考格式（基于厦门素材）

```
【政策名称】[政策名称]

【发布机构】[机构名称]

【发布时间】[时间]

【政策类型】
[产业/数据/AI/新质生产力等]

【核心内容】
- 内容1
- 内容2
- 内容3

【支持措施】
- 措施1：具体说明
- 措施2：具体说明
- 措施3：具体说明

【数据指标】
- 指标1：目标
- 指标2：目标
- 指标3：目标
```

### 整理要点

1. **表格形式**：核心内容、支持措施、数据指标用表格或列表
2. **量化数据**：包含具体数字（金额、比例、数量）
3. **分类清晰**：按政策类型分类
4. **措施具体**：每个措施有具体说明

### 示例（思明区AI政策）

```
【政策名称】厦门市思明区促进新一代人工智能产业发展若干措施

【发布机构】厦门市思明区人民政府

【发布时间】2024年12月23日

【政策类型】
人工智能产业政策

【核心内容】
- 每年安排1亿元政策扶持资金
- 提出10条扶持措施
- 推动AI产业向"新"、发展提"质"

【支持措施】
- 建设AI产业公共服务平台
- 创新"AI券"服务模式
- 支持企业打造标杆应用场景
- 鼓励企业参与标准制定

【数据指标】
- 人工智能产业链上下游企业达308家
- 每年举办活动：2024年11场、2025年13场
- 培训人才：2024年821人次、2025年713人次
```

## 五、不同类型信息的检索重点

### 政府官员

| 检索项 | 说明 |
|--------|------|
| 现任职务 | 当前担任的具体职务 |
| 履历信息 | 以往担任的重要职务 |
| 工作重点 | 当前关注的领域 |
| 近期动向 | 最近3个月的重要活动 |
| 重点关注领域 | 长期关注的专业领域 |

### 企业信息

| 检索项 | 说明 |
|--------|------|
| 企业简介 | 一句话概括 |
| 行业地位 | 在行业中的位置 |
| 核心业务 | 主要业务范围 |
| 技术能力 | 核心技术或产品 |
| 近期动态 | 近期重要进展 |

### 高校/科研机构

| 检索项 | 说明 |
|--------|------|
| 机构定位 | 国家级/省级/市级 |
| 研究方向 | 主要研究领域 |
| 科研成果 | 重大成果或奖项 |
| 核心资源 | 数据、人才、设备等 |

## 六、信息输出到谈参的位置

### 人物信息输出到"关键人介绍"

系统会将检索到的人物信息整理后，插入到谈参的"四、关键人介绍"部分。

### 企业信息输出到"对方情况"

系统会将检索到的企业信息整理后，插入到谈参的"三、对方情况"部分。

### 政策信息输出到"地区合作基础"

系统会将检索到的政策信息整理后，插入到谈参的"二、地区合作基础"的"政策环境"部分。

---

## 七、信息验证与标注

### 已验证信息

- 系统检索到且有明确来源的信息
- 标注"基于公开资料"

### 待验证信息

- 需要进一步核实的信息
- 标注"待确认"

### 缺失信息

- 未检索到的信息
- 标注"待补充"

---

## 八、敏感信息过滤

系统在整理信息时，会过滤以下类型的信息：

### 政治敏感

- 不涉及政治争议
- 不评论政治立场
- 不涉及敏感政治人物

### 商业敏感

- 不透露未公开的财务数据
- 不涉及具体的商业机密
- 不评论竞争对手

### 个人隐私

- 不涉及个人家庭信息
- 不涉及个人联系方式
- 不涉及敏感个人经历

## 十一、信息更新机制

### 持续学习

系统会根据用户提供的新素材，不断学习和优化信息整理风格。

### 知识库更新

每次补充素材后，系统会将新信息整理并添加到相应的知识库文件中。

### 风格一致性

系统会始终保持与已有素材一致的整理风格和格式标准。
