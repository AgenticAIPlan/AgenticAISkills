# 重要会谈谈参生成助手

> 重要会谈谈话参考生成 Skill - 面向部门领导，在重要客户/合作伙伴/高校/政府/生态伙伴会谈前，快速生成可上桌使用的背景简报、合作判断、谈话要点与后续推进建议。

## Skill定位

本 Skill 不是一个简单的"写材料助手"，而是基于"对象类型识别 + 地区维度分析 + 双方关系建模 + 合作成熟度判断 + 战略目标拆解"的分析框架，将分散的会谈背景信息转化为结构化的谈参内容，支持高层会谈的高效决策与表达。

## 适用场景

### 按对象类型

| 会谈对象类型 | 说明 |
|-------------|------|
| 企业客户/伙伴 | 商业公司、技术合作伙伴、生态伙伴、AI创业团队 |
| 高校/科研机构 | 国家重点实验室、高校科研团队、研究院所 |
| 政府/协会 | 政府部门、行业协会、产业联盟、示范园区 |
| 开源社区 | 开源项目、国际标准组织、开发者联盟 |

### 按地区维度

| 地区类型 | 代表城市 | 负责人 | 业务特色 |
|----------|----------|--------|----------|
| 中心城市 | 深圳、成都、厦门 | 华苒君、王宇、于明睿 | 中心建设、生态运营、产业落地 |
| 政府合作 | 长沙、南昌 | 刘可萌、刘芸 | 国资合作、政策支持、示范项目 |
| 产业特色 | 武汉、东莞 | 刘可萌、郑越 | 工业应用、制造业升级、智慧城市 |
| 国际窗口 | 香港 | 华苒君 | 国际合作、数据引入、品牌出海 |

【会谈时间/地点】[时间+地点]

【对方简介】
[简要描述对方是谁、做什么的]

[如果有，简述合作内容]

[对方想什么、要什么]

[本次会谈的目标]

## 核心方法论

### 双维度分析框架

**对象类型 × 地区维度** 决定谈参结构：

| 对象\地区 | 中心城市 | 政府合作 | 产业特色 | 国际窗口 |
|-----------|----------|----------|----------|----------|
| 企业 | 中心+产业 | 政策+项目 | 应用+案例 | 国际合作 |
| 高校 | 产学研 | 示范项目 | 技术应用 | 数据引入 |
| 政府 | 中心建设 | 政策支持 | 产业升级 | 国际交流 |

### 合作成熟度模型

```
阶段1：技术验证 → 阶段2：产品共创 → 阶段3：落地案例 → 阶段4：生态协同 → 阶段5：商业化
```

### 领导话术三段式

1. **认可对方**（建立关系）
2. **强调已有合作**（建立信任）
3. **引出未来合作**（推动行动）

## 更新建议

### 持续补充

建议持续更新以下内容：

- **新增案例**：每次合作完成后补充到 CASES.md
- **新增能力**：产品/能力更新后补充到 CAPABILITIES.md
- **更新话术**：发现有效表达后补充到 TALKING_POINTS.md

### 质量标准

- 案例需包含：合作背景、地区、负责人、成熟度、内容、亮点、话术、后续推进
- 能力需包含：标签、适用场景、对应产品、典型案例
- 话术需包含：中英文版本、适用对象、使用场景
- 地区需包含：负责人、业务定位、政策环境、核心合作、业务特色
