# ERNIE-Image 提示词撰写指南

## 概述

ERNIE-Image-Turbo 是百度星河社区的图像生成模型，对中文 Prompt 支持良好，能够准确理解中文语义和文化背景。

---

## 核心原则

### 1. 中文优先

ERNIE-Image 对中文 Prompt 效果最佳：

✅ 推荐：
```
一位科学家在实验室里研究黑洞图像，背景是复杂的仪器设备
```

⚠️ 可用但不推荐：
```
A scientist studying black hole images in a laboratory with complex equipment
```

### 2. 结构化描述

采用 **主体 → 动作 → 环境 → 风格** 的顺序：

```
[主体描述]，[动作/状态]，[环境背景]，[风格关键词]
```

**示例**：
```
一只可爱的抗体角色（Y字形，戴着小警帽），正在追逐一个滑稽的病毒角色（绿色小球，带尖刺），人体内部环境，卡通科普风格，柔和粉彩配色
```

### 3. 具体胜于抽象

❌ 抽象描述：
```
一个美丽的宇宙场景
```

✅ 具体描述：
```
深邃的宇宙空间，中心是一个巨大的黑洞，橙红色的吸积盘环绕，背景是紫色的星云和闪烁的星辰
```

## 常见问题与解决方案

### 问题1：风格不一致

**症状**：多张图看起来风格不统一

**解决方案**：
1. 确保 `style_seed` 原文追加到每个 Prompt 末尾
2. 使用固定的风格关键词组合
3. 避免在单个 Prompt 中混合冲突的风格词

**示例**：
```python
# 统一风格追加
style_seed = "扁平插画风格，清晰轮廓，科学配色"
final_prompt = f"{image_prompt}，{style_seed}，高质量，连环画"
```

### 问题2：人物变形

**症状**：人物比例不协调、肢体异常

**解决方案**：
1. 添加比例约束词
2. 简化人物姿势描述
3. 避免复杂的多人物场景

**改进示例**：
```
# 原提示
一群科学家在实验室讨论

# 改进提示
一位戴眼镜的科学家，站立姿势，双手自然下垂，正在与画面外的同事交流，单人居中构图，比例协调
```

### 问题3：细节缺失

**症状**：重要元素未出现或不够清晰

**解决方案**：
1. 明确列出需要的关键元素
2. 使用"带有"、"显示"等指令词
3. 增加元素的具体描述

**改进示例**：
```
# 原提示
科学家在看屏幕

# 改进提示
科学家面对大型显示屏，屏幕上清晰显示橙红色的黑洞光环图像，图像中心是黑色的事件视界，周围有明亮的光环
```

### 问题4：构图混乱

**症状**：画面元素分布不清晰，主题不突出

**解决方案**：
1. 添加构图指令
2. 明确前景/背景关系
3. 指定画面焦点

**构图关键词**：
```
居中构图、三分法构图、对称构图、前景聚焦、背景虚化
```

## 提示词长度建议

### 最佳长度

- **单 Panel**：80-150 中文字符
- **全局合成**：200-400 中文字符

### 过短风险

```
# 太短，信息不足
一个黑洞

# 改进
超大质量黑洞，中心漆黑的事件视界被橙红色吸积盘环绕，背景是璀璨的宇宙空间，扁平插画风格
```

### 过长风险

```
# 太长，信息过载
一个巨大无比的超大质量黑洞位于画面的正中央位置，它的中心是一个非常非常漆黑的事件视界，周围环绕着一圈又一圈橙红色的吸积盘，吸积盘发出明亮的光芒，背景是深邃无垠的宇宙空间，里面点缀着数不清的星星和五颜六色的星云，远处还有几个模糊的星系...

# 改进（精简到关键元素）
超大质量黑洞，中心漆黑的事件视界，橙红色发光吸积盘，深邃宇宙背景点缀星辰，扁平插画风格，高质量
```

## 审核规避指南

### 避免的词汇

| 类别 | 避免词 | 替代方案 |
|-----|-------|---------|
| 暴力 | 血、伤口、战斗 | 使用示意图、卡通化处理 |
| 敏感 | 政治人物、敏感事件 | 使用通用的科学家形象 |
| 商标 | 具体品牌名 | 使用通用描述（如"智能手机"） |

乔布斯发布iPhone

## 进阶技巧

### 负面提示词（部分场景可用）

虽然 ERNIE-Image 不直接支持负面提示词，但可以通过正面描述规避：

```
# 避免模糊
清晰锐利，细节丰富，不模糊

# 避免变形
比例协调，人体结构正确
```

### 多语言混合（谨慎使用）

专有名词可保留英文：

```
DNA双螺旋结构，E=mc²公式，NASA风格宇航服
```

### 迭代优化流程

1. **初版**：基础描述
2. **评估**：检查缺失元素
3. **迭代**：补充细节、调整风格
4. **定型**：确认效果满意

```
# 迭代示例

# V1
科学家看显微镜

# V2
一位女科学家正在使用显微镜观察样本，实验室背景

# V3（最终）
一位戴眼镜的女科学家，穿着白色实验服，正专注地通过显微镜观察，显微镜下的样本发出淡淡的蓝光，实验室背景有整齐的器材架，科学插画风格，高质量
```
