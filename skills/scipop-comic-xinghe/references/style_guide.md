# 连环画风格指南

## 风格种子（style_seed）设计原则

### 什么是 style_seed？

`style_seed` 是一个简短的风格描述字符串，用于确保所有 Panel 的视觉风格一致。它在 Phase 1 生成后，会在后续所有图像生成调用中原样追加。

### 设计原则

1. **简洁性**：控制在 15-30 个中文字符以内
2. **具体性**：描述具体的视觉特征而非抽象概念
3. **一致性**：风格元素之间不冲突

### 风格维度

| 维度 | 示例描述 |
|-----|---------|
| **画风** | 扁平插画、手绘素描、水彩风格、赛博朋克 |
| **色调** | 柔和粉彩、深邃太空蓝、霓虹光效、复古暖色 |
| **线条** | 清晰轮廓、粗犷笔触、细腻描边 |
| **质感** | 2D矢量艺术、纸质感、数字艺术 |
| **氛围** | 科学严谨、可爱治愈、神秘奇幻 |

## 常用风格模板

### 科学严谨风

```
扁平插画风格，清晰轮廓，专业科学配色，2D矢量艺术
```

适用：天文学、物理学、生物学等硬科普

### 可爱治愈风

```
可爱卡通风，柔和粉彩配色，圆润造型，温馨友好
```

适用：儿童科普、健康知识、日常科学

### 科技未来风

```
赛博朋克风格，霓虹光效，深色背景，高科技感
```

适用：人工智能、量子计算、前沿科技

### 自然清新风

```
水彩插画风格，清新自然色调，柔和笔触，生态感
```

适用：环境科学、生态科普、地理知识

### 复古怀旧风

```
复古连环画风，暖黄怀旧色调，经典漫画风格
```

适用：科学史、历史科普、传统文化

## 场景描述（scene）撰写

### 结构模板

```
[主体] + [动作/状态] + [环境/背景] + [情绪/氛围]
```

### 示例

| 好的描述 | 问题描述 |
|---------|---------|
| 科学家在实验室中激动地盯着屏幕上的黑洞图像，背景是复杂的仪器设备 | 科学家看屏幕（太简单） |
| 一个可爱的抗体角色（Y字形，戴小警帽）正在追逐滑稽的病毒角色 | 抗体打病毒（缺少细节） |

## 图像提示（image_prompt）撰写

### 核心要素

1. **主体描述**：主要角色或物体
2. **动作姿态**：正在发生什么
3. **环境背景**：场景设置
4. **风格指令**：与 style_seed 协调
5. **技术参数**：画质、构图等

### 中文Prompt优势

Ernie-image-turbo 对中文 Prompt 支持良好：
- 可以直接描述中国元素、东方审美
- 专有名词翻译更准确
- 文化背景理解更深入

### 禁忌词避免

避免使用可能导致审核拒绝的词汇：
- 暴力、血腥相关
- 政治敏感内容
- 商标品牌（可用通用描述替代）

## Panel 数量决策

### 4格适用场景

- 内容结构清晰，可划分为4个阶段
- 信息量适中，每格承载一个核心概念
- 适合故事性强、有明确起承转合的内容

### 5格适用场景

- 内容有5个层次或要点
- 需要一个过渡或高潮格
- 适合复杂流程或多层次解释

### 6格适用场景

- 内容丰富，知识点多
- 需要展示完整过程（如实验步骤）
- 适合系统性强的科普内容

## 全局合成注意事项

### 布局选择

| Panel数 | 推荐布局 | 视觉效果 |
|--------|---------|---------|
| 4 | 2×2 | 正方形，均衡 |
| 5 | 2×3 | 横向阅读，留白收尾 |
| 6 | 2×3 | 标准竖向长图 |

### 阅读顺序

中文连环画采用**从左到右、从上到下**的阅读顺序：
```
┌─────┬─────┐
│  1  │  2  │
├─────┼─────┤
│  3  │  4  │
├─────┼─────┤
│  5  │  6  │
└─────┴─────┘
```
