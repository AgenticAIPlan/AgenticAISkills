# 文章解析提示词模板

## Phase 1 系统提示词

### 基础模板

```
你是一位科普连环画脚本编写专家。分析文章并根据内容的丰富程度和结构确定最合适的 Panel 数量（4-6个）。输出一个 JSON 对象，包含四个字段："recommended_panels"（整数，4-6），"recommendation_reason"（一句话中文解释为什么这个 Panel 数量适合该文章），"style_seed"（简短的中文风格描述，在所有 Panel 中复用），"panels"（与推荐数量匹配的对象数组，每个对象包含 "id"、"scene" 中文场景描述、"image_prompt" 中文图像生成提示）。仅输出原始 JSON，不要使用 markdown 代码块。
```

### 进阶模板（带风格预设）

```
你是一位科普连环画脚本编写专家。

## 任务
分析科普文章，拆解为 4-6 个视觉场景，输出连环画脚本。

## 风格预设
根据文章主题选择合适的风格：
- 天文/物理：深邃太空风格，科技蓝紫配色，星云光效
- 生物/医学：可爱卡通风，柔和粉彩，圆润角色设计
- 环境/生态：水彩自然风，清新绿调，有机质感
- 科技/AI：赛博朋克风，霓虹光效，高科技感
- 科学史：复古连环画风，怀旧暖色调

## 输出格式
仅输出 JSON，不要使用 markdown 代码块：
{
  "recommended_panels": <4-6>,
  "recommendation_reason": "<一句话中文解释>",
  "style_seed": "<简短中文风格描述>",
  "panels": [
    {
      "id": <序号>,
      "scene": "<中文场景描述>",
      "caption": "<科普旁白，20字以内>",
      "image_prompt": "<中文图像生成提示>"
    }
  ]
}
```

---

## 场景设计原则

### 核心原则

1. **一图一义**：每个 Panel 只承载一个核心知识点或叙事节点
2. **视觉可绘**：场景必须是"可以画出来的"，避免纯抽象概念
3. **叙事连贯**：Panel 之间有逻辑递进或时间顺序
4. **节奏平衡**：信息密度分布均匀，避免某格过于拥挤

---

### 文章类型识别与拆分策略

#### 类型一：叙事型文章

**特征**：有故事线、时间线、人物或事件发展过程

**拆分策略**：按时间轴/事件阶段拆分

```
Panel 1: 起因/背景引入
Panel 2: 发展/问题出现
Panel 3: 关键转折/高潮
Panel 4: 结果/影响
Panel 5: 总结/延伸（可选）
Panel 6: 展望/启发（可选）
```

**示例**：《人类首次拍摄黑洞照片》
```
Panel 1: 科学家们围坐在会议室，讨论拍摄黑洞的挑战
Panel 2: 全球望远镜联网协作的壮观场景
Panel 3: 数据处理过程中海量的硬盘和计算机
Panel 4: 历史性时刻——黑洞照片首次亮相
Panel 5: 科学意义解读——验证爱因斯坦理论
Panel 6: 未来展望——更清晰的黑洞影像
```

---

#### 类型二：知识点型文章

**特征**：多个并列或递进的知识点，无明确叙事线

**拆分策略**：按知识点数量拆分，每个知识点一图

```
Panel 1: 核心概念引入（"是什么"）
Panel 2-N: 知识点展开（"为什么/怎么样"）
Panel N+1: 总结/应用场景（可选）
```

**示例**：《人体免疫系统的三大防线》
```
Panel 1: 免疫系统全貌——身体内部的"防御网"
Panel 2: 第一道防线——皮肤和黏膜的物理屏障
Panel 3: 第二道防线——巨噬细胞的吞噬作用
Panel 4: 第三道防线——抗体的精准打击
Panel 5: 疫苗的作用——训练免疫系统
```

---

#### 类型三：流程原理型文章

**特征**：描述工作原理、操作步骤、循环过程

**拆分策略**：按流程阶段拆分，强调因果链

```
Panel 1: 整体系统/过程概览
Panel 2-N: 关键步骤逐一展示
Panel N+1: 结果/输出（可选）
```

**示例**：《疫苗如何保护我们》
```
Panel 1: 疫苗进入人体——一个带有笑脸的小瓶子
Panel 2: 免疫细胞识别疫苗——"侦察兵"发现入侵者
Panel 3: 抗体产生——细胞工厂开始生产武器
Panel 4: 记忆细胞形成——"档案库"储存敌人信息
Panel 5: 真实病毒来袭——抗体迅速响应消灭敌人
Panel 6: 身体恢复健康——胜利的免疫系统
```

---

#### 类型四：对比论证型文章

**特征**：多个观点、现象、数据的对比分析

**拆分策略**：对比双方/多方分格展示，强调差异

```
Panel 1: 对比主题引入
Panel 2: A 方案/观点展示
Panel 3: B 方案/观点展示
Panel 4: 对比结果/结论
Panel 5-6: 延伸分析（可选）
```

**示例**：《核能 vs 太阳能：谁更清洁？》
```
Panel 1: 能源选择的难题——天平两端的思考
Panel 2: 核能的优势——高能量密度，低碳排放
Panel 3: 核能的挑战——废料处理与安全风险
Panel 4: 太阳能的优势——清洁可再生
Panel 5: 太阳能的挑战——受天气影响，储能需求
Panel 6: 结论——因地制宜的能源组合
```

---

#### 类型五：探索发现型文章

**特征**：科学发现过程、研究突破、实验结果

**拆分策略**：按"问题→探索→发现→意义"展开

```
Panel 1: 科学问题提出/研究动机
Panel 2: 研究方法/实验设计
Panel 3: 关键发现/突破时刻
Panel 4: 数据/证据展示
Panel 5: 科学意义解读
Panel 6: 未来研究方向（可选）
```

**示例**：《发现系外行星》
```
Panel 1: 千年追问——宇宙中还有其他家园吗？
Panel 2: 凌日法观测——望远镜盯着星星眨眼睛
Panel 3: 数据异常——恒星亮度的微小变化
Panel 4: 确认发现——系外行星的艺术想象图
Panel 5: 宜居带概念——这个行星可能存在液态水
Panel 6: 寻找第二个地球——未来的太空望远镜
```

### Panel 位置设计详解

#### Panel 1：开篇定位

**功能**：吸引注意、建立情境、提出问题

**设计要点**：
- 视觉冲击力：使用醒目的主体或反差场景
- 情绪钩子：好奇、惊讶、悬念
- 清晰的"谁/什么/在哪里"

**好的开篇示例**：
```
❌ "介绍黑洞"
✅ "一个巨大的橙色光环漂浮在黑暗太空中，中心是完全漆黑的圆形区域——这就是人类首次拍摄到的黑洞照片"
```

**常见错误**：
- 信息过载：试图在第一格塞入所有背景
- 过于抽象：没有具体视觉元素
- 缺乏情绪：平铺直叙，没有吸引力

---

#### Panel 2-3（4格）/ Panel 2-4（5-6格）：主体展开

**功能**：展开论述、层层深入、核心内容

**设计要点**：
- 每格聚焦一个子主题
- 避免信息重复
- 保持叙事节奏：问题→分析→过程→细节

**展开模式**：

| 模式 | Panel 2 | Panel 3 | Panel 4（如有） |
|-----|---------|---------|----------------|
| **递进式** | 问题深化 | 原因分析 | 解决方案 |
| **并列式** | 方面 A | 方面 B | 方面 C |
| **因果式** | 现象描述 | 原因解释 | 结果展示 |
| **时空式** | 早期阶段 | 中期发展 | 当前状态 |

**展开阶段示例**：
```
Panel 2: 为什么光无法逃逸黑洞？（原理解释）
Panel 3: 科学家如何"看到"黑洞？（观测方法）
Panel 4: 全球望远镜联网协作的场景（技术细节）
```

---

#### Panel 4（4格）/ Panel 5-6（5-6格）：收尾升华

**功能**：总结结论、延伸思考、情感升华

**设计要点**：
- 回应开篇：形成闭环
- 延伸价值：启发思考或行动号召
- 情感落点：让读者记住什么

**收尾模式**：

| 模式 | 特点 | 示例 |
|-----|------|------|
| **总结型** | 概括核心要点 | "所以，黑洞教会我们..." |
| **展望型** | 未来发展方向 | "下一个目标是..." |
| **启发型** | 引发更深思考 | "如果黑洞存在，那么..." |
| **行动型** | 呼吁读者行动 | "保护环境，从..." |

**好的收尾示例**：
```
❌ "黑洞研究很重要"
✅ "一位年轻的科学家仰望星空，远处是璀璨的银河，旁白：'宇宙还有无数秘密等待我们去发现'"
```

### 场景过渡技巧

#### 时间过渡

```
Panel 1: 早晨的太阳系俯视图
Panel 2: 几亿年后的太阳系——太阳变成红巨星
→ 用视觉变化暗示时间流逝
```

#### 空间过渡

```
Panel 1: 宏观宇宙——银河系
Panel 2: 放大镜效果——太阳系
Panel 3: 继续放大——地球
→ 用"放大/缩小"效果连接不同尺度
```

#### 因果过渡

```
Panel 1: 工厂排放大量二氧化碳
Panel 2: 大气层中温室气体累积（箭头指向地球）
Panel 3: 全球气温上升的温度计图示
→ 用箭头、流程图等视觉元素串联因果关系
```

#### 对比过渡

```
Panel 1: 左侧：传统汽油车冒烟
Panel 2: 右侧：电动车安静行驶
→ 用分屏、镜像等布局强化对比
```

### 场景描述（scene）撰写规范

#### 结构模板

```
[主体] + [动作/状态] + [环境背景] + [情绪氛围]
```

#### 撰写要点

| 维度 | 要点 | 示例 |
|-----|------|------|
| **主体** | 明确谁/什么 | "一位戴眼镜的女科学家" |
| **动作** | 具体在做什么 | "正激动地用手指着屏幕" |
| **环境** | 在哪里 | "现代实验室，有显微镜和电脑" |
| **情绪** | 什么感受 | "专注、兴奋、紧张" |
| **细节** | 关键元素 | "屏幕显示黑洞光环图像" |

#### 好的场景描述对比

| ❌ 问题描述 | ✅ 好的描述 |
|-----------|-----------|
| 科学家看屏幕 | 一位戴眼镜的女科学家，穿着白色实验服，正激动地盯着面前的电脑屏幕，嘴巴微张，眼中闪烁着兴奋的光芒 |
| 抗体打病毒 | 一个白色的Y字形抗体角色，戴着蓝色小警帽，正伸出双臂追逐前方滑稽逃跑的绿色球形病毒 |
| 黑洞图片 | 深邃的太空中，一个巨大的橙红色光环包围着漆黑的圆形中心，周围点缀着遥远的星星 |

### caption（科普旁白）撰写规范

**字数限制**：20字以内，简洁有力

**功能定位**：辅助理解、点睛之笔、记忆锚点

**撰写技巧**：

| 技巧 | 示例 |
|-----|------|
| **数字锚定** | "首张黑洞照片诞生" |
| **概念定义** | "事件视界：光的终结点" |
| **因果总结** | "温室效应导致全球变暖" |
| **情感共鸣** | "人类探索宇宙的一小步" |
| **提问启发** | "宇宙中还有其他生命吗？" |

**常见错误**：
- 过长："科学家们经过多年的努力终于拍摄到了人类历史上第一张黑洞照片"（太长）
- 空洞："这是一个重要的发现"（无信息量）
- 重复：与图像内容完全重复

### 常见问题与解决方案

| 问题 | 原因 | 解决方案 |
|-----|------|---------|
| **场景过于抽象** | 概念无法直接可视化 | 使用比喻、拟人、场景化转换 |
| **Panel 之间缺乏连贯性** | 拆分过于机械 | 添加过渡元素（时间、空间、因果） |
| **信息分布不均** | 某个知识点过于复杂 | 将复杂知识点拆分为多格 |
| **开篇平淡无奇** | 缺乏视觉钩子 | 使用冲突、悬念、震撼场景 |
| **结尾戛然而止** | 缺乏升华 | 添加总结、展望、启发思考 |
| **caption 超出字数** | 试图塞入太多信息 | 精简为核心要点，20字以内 |
| **image_prompt 不够具体** | 缺乏细节描述 | 参见五段式模板（主体→动作→环境→细节→风格） |
```

## 输出 JSON 结构详解

### recommended_panels

**决策逻辑**：

| Panel数 | 适用场景 |
|--------|---------|
| 4 | 内容简洁，结构清晰，有明确的起承转合 |
| 5 | 内容适中，需要一个过渡或高潮格 |
| 6 | 内容丰富，知识点多，需要完整叙事链 |

### recommendation_reason

**示例**：
- "文章涵盖5个核心知识点，适合5格展示。"（知识密度型）
- "故事有明确的起承转合四阶段，4格最佳。"（叙事结构型）
- "内容包含详细的实验步骤，6格可完整呈现。"（流程展示型）

### style_seed

**格式**：`{画风}，{色调}，{线条/质感}，{氛围}`

**示例**：
```
扁平插画风格，深邃太空蓝配色，清晰轮廓，科学严谨氛围
可爱卡通风，柔和粉彩，圆润造型，温馨友好
水彩手绘风，清新自然色调，柔和笔触，生态感
赛博朋克风格，霓虹光效，高科技质感，未来感
```

### panels[].scene

**格式**：`[主体] + [动作/状态] + [环境] + [情绪]`

**示例**：
```
一位天文学家在控制室激动地看着屏幕上的黑洞首张照片，背景是复杂的仪器设备和欢呼的同事
```

### panels[].image_prompt

**结构**：
1. **主体描述**：主要角色或物体的具体描述
2. **动作姿态**：正在发生什么
3. **环境背景**：场景设置
4. **细节元素**：关键的视觉元素
5. **风格关键词**：与 style_seed 呼应

**示例**：
```
一位戴眼镜的女科学家，穿着白色实验服，兴奋地指着大型显示屏，屏幕上显示橙红色的黑洞光环图像，周围有其他科学家欢呼庆祝，实验室背景，各种仪器设备，扁平插画风格，清晰轮廓，科学严谨配色
```

## 提示词优化技巧

### 增强主体清晰度

❌ 弱提示：
```
一个科学家在做实验
```

✅ 强提示：
```
一位戴眼镜的中年男科学家，穿着白色实验服和蓝色手套，正在用移液管向试管中添加蓝色液体，专注的表情，实验室背景有显微镜和培养皿
```

### 增强场景氛围

❌ 弱提示：
```
黑洞的照片
```

✅ 强提示：
```
深空中神秘的黑洞，中心是漆黑的事件视界，周围是橙红色的吸积盘光环，背景是璀璨的星云和遥远的星系，科幻感，深邃神秘
```

### 增强风格一致性

在每个 `image_prompt` 末尾添加：
```
，{style_seed}，高质量，连环画
```

示例：
```
一位宇航员在月球表面插旗，背景是蓝色的地球悬挂在黑色天空中，扁平插画风格，清晰轮廓，科学严谨配色，高质量，连环画
```
