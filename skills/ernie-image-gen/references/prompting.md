# ERNIE Image Prompting 指南

ERNIE Image 的优势重点是：

- 中文指令遵循
- 多语言文本渲染
- 长文本
- 多区域文本布局

因此 prompt 建议按以下结构组织。

## 通用模板

```text
任务类型：文生图
主体：<主要对象>
场景：<环境/地点/背景>
风格：<摄影/插画/海报/3D/国风/写实>
构图：<近景/中景/远景/俯视/留白/海报排版>
光线与氛围：<自然光/棚拍/电影感/高对比/柔和>
细节要求：<材质/颜色/表情/动作/镜头语言>
文本语言：<中文/英文/日文/中英混排/中日混排>
图中文字（逐字）：<必须原样出现的文字，没有则留空>
文字布局：<标题/副标题/左上角/居中/多区域分布>
约束：<不要出现什么；哪些元素必须保留>
```

## 普通中文文生图

适合直接用完整自然语言描述：

```text
一只戴宇航员头盔的橘猫坐在飞船驾驶舱内，窗外是蓝紫色星云，电影感打光，超高细节，金属与玻璃材质真实，构图稳定，中文海报级质感
```

## 含图中文字

涉及文字时，把“逐字内容”单独强调，不要埋在长句里。

推荐结构：

```text
海报设计，一杯冰美式放在木桌上，清晨阳光，极简咖啡品牌风格。
图中文字（逐字）：
1. 今日特调
2. Cold Brew
3. 限时供应
文字布局：标题居上，英文副标题在中部偏右，活动文案放底部横条内。
要求文字清晰可读，不要错字，不要多余字符。
```

## 多语言 / 中英日混合

推荐显式标注每块文本所属语言：

```text
日式甜品店宣传海报，暖色灯光，精致包装摄影风格。
图中文字（逐字）：
中文：春日限定
英文：Spring Matcha Collection
日文：抹茶の香り
文字布局：中文主标题居中，英文副标题在下方，日文点缀放右上角。
要求三种语言都清晰可读，排版协调。
```

## 长文本 / 多区域文本

长文本最好拆成编号块，明确“每块的相对位置”。

```text
产品详情页风格的宣传图，主体是一支银色保温杯，白底，简洁电商视觉。
图中文字（逐字）：
1. 标题：轻量保温，一杯随行
2. 卖点1：316不锈钢内胆
3. 卖点2：12小时长效保温
4. 卖点3：一键开合，防漏设计
文字布局：
- 标题位于上方居中
- 三条卖点分布在右侧三段区域
要求文字准确、分区清晰、信息层级明确。
```

## 建议的 agent 行为

- 默认中文优先
- 只要图里有字，就单独抽出“图中文字（逐字）”
- 长文本默认拆块
- 如果用户没给布局，就自动补“标题 / 副标题 / 辅助卖点”这种安全结构
- 如果用户要求复现品牌文案，必须逐字照抄，不要擅自改写
