## 扩展矩阵

如需添加新的场景或硬件，在 `src/pages/AllCases.tsx` 的 `HARDWARE_COMPATIBILITY` 对象中添加：

```typescript
'新场景': ['硬件A', '硬件B', '硬件C'],
```

## 使用流程

1. 用户点击"硬件兼容分析"按钮
2. 选择一个场景（如"LLM推理"）
3. 系统显示该场景推荐的硬件列表
4. 用户可点击"筛选案例"查看该硬件的所有相关案例
