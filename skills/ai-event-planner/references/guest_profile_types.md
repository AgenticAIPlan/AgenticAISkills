# 嘉宾画像类型参考

## 专业背景分类

### 学术界
- **大学教授/副教授**
  - 研究方向：深度学习、NLP、CV等AI核心领域
  - 成果：顶级会议论文发表、专利持有
  - 适合环节：主题演讲（理论基础、前沿趋势）

- **研究院研究员**
  - 研究机构：中科院、微软亚洲研究院、高校AI实验室等
  - 专长：算法研究、技术创新
  - 适合环节：主题演讲、圆桌讨论

### 工业界
- **CTO/首席技术官**
  - 背景：大型科技公司或AI独角兽
  - 关注：技术战略、产品落地、商业化
  - 适合环节：主题演讲（实践经验、行业洞察）

- **技术总监/技术VP**
  - 背景：知名企业AI部门负责人
  - 专长：团队管理、技术架构、规模化应用
  - 适合环节：圆桌讨论、案例分享

- **高级算法工程师**
  - 背景：一线技术专家
  - 专长：具体技术实现、工程落地
  - 适合环节：Demo展示、技术分享

- **产品经理（AI方向）**
  - 背景：AI产品负责人
  - 专长：产品设计、用户需求、商业化
  - 适合环节：圆桌讨论（产品视角）

### 创业者
- **AI创业公司创始人**
  - 背景：AI垂直领域创业者
  - 专长：创新应用、市场洞察
  - 适合环节：主题演讲（创业故事、产品创新）

## 职位级别分类

### 高层决策者
- CTO、CEO、创始人、院长
- 特点：战略视野、行业影响力
- 适合：开幕式致辞、主题演讲、圆桌主持

### 中层管理者
- 技术总监、VP、产品总监
- 特点：实践经验、管理经验
- 适合：主题演讲、圆桌讨论、案例分享

### 技术专家
- 首席科学家、研究员、资深工程师
- 特点：技术深度、前沿研究
- 适合：技术演讲、Demo展示、圆桌讨论

## 领域专长分类

### 算法研究类
- 深度学习、强化学习
- NLP、CV、多模态
- 模型训练、优化、部署

### 工程落地类
- 大规模模型训练
- 推理优化、工程化
- 边缘计算、端侧AI

### 产品应用类
- AI产品设计
- 商业化应用
- 用户体验

### 行业垂直类
- 医疗AI专家
- 金融科技专家
- 教育科技专家
- 自动驾驶专家
