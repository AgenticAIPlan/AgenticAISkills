# Example Business Skill

## 适用场景

当用户需要处理某个明确的业务任务，且该任务需要固定的执行步骤、输入约束和输出结构时，使用本 Skill。

## 输入要求

- 业务目标
- 背景上下文
- 关键约束
- 期望输出形式

## 执行步骤

1. 先确认业务目标和边界条件。
2. 提取执行所需的关键输入信息。
3. 按固定流程完成任务处理。
4. 对结果进行自查，确保满足格式和质量要求。

## 输出要求

- 输出结构清晰
- 结论与过程一致
- 明确指出风险、假设和待确认项
