## 情感分析概述

### 定义

情感分析（Sentiment Analysis）是指通过自然语言处理技术，识别和提取文本中的主观信息，判断文本所表达的情感倾向。

在文心大模型监测场景下，情感分析用于：
- 了解用户对文心大模型的态度
- 发现产品问题和改进方向
- 识别潜在的舆情风险
- 评估用户满意度变化趋势

### 分析对象

| 分析对象 | 说明 | 处理方式 |
|---------|------|----------|
| 文章正文 | 主要内容分析 | 全文分析 |
| 评论内容 | 用户反馈分析 | 单条分析+汇总 |
| 标题 | 快速情感判断 | 简要分析 |
| 话题标签 | 情感倾向推断 | 上下文分析 |

## 情感分类体系

### 一级分类

```
情感倾向
├── 正向 (Positive)
├── 负向 (Negative)
├── 中性 (Neutral)
└── 混合 (Mixed)
```

### 二级分类

#### 正向情感细分

| 二级分类 | 定义 | 典型词汇 |
|---------|------|----------|
| 赞赏 | 对能力或成就的认可 | 优秀、出色、厉害、强大 |
| 推荐 | 建议他人使用 | 推荐、值得一试、值得拥有 |
| 满意 | 对体验感到满意 | 满意、惊喜、超出预期 |
| 信任 | 对产品有信心 | 可靠、值得信赖、稳定 |
| 喜欢 | 个人喜爱 | 喜欢、爱用、离不开 |

#### 负向情感细分

| 二级分类 | 定义 | 典型词汇 |
|---------|------|----------|
| 批评 | 对不足之处的批评 | 不好、差劲、落后 |
| 失望 | 未达到期望 | 失望、遗憾、不如预期 |
| 质疑 | 对真实性的怀疑 | 怀疑、假的、吹牛 |
| 担忧 | 对未来的担心 | 担心、忧虑、前景不明 |
| 抱怨 | 对问题的抱怨 | 投诉、不满、吐槽 |

#### 中性情感细分

| 二级分类 | 定义 | 典型特征 |
|---------|------|----------|
| 客观描述 | 事实陈述，无情感 | 数据、参数、功能介绍 |
| 中立观点 | 平衡正反面 | 既有优点也有缺点 |
| 疑问询问 | 提出问题 | 如何、怎么、是否 |

#### 混合情感细分

| 类型 | 定义 | 示例 |
|------|------|------|
| 正负并存 | 同时包含正负面 | "功能很强大，但是价格有点贵" |
| 先正后负 | 先表扬后批评 | "界面很漂亮，就是有时候会卡" |
| 先负后正 | 先批评后表扬 | "刚开始不太会用，熟练后觉得很好" |

### 情感强度

| 强度 | 定义 | 特征 |
|------|------|------|
| 极强 | 情感表达非常强烈 | 使用最高级、感叹号、重复词 |
| 强 | 情感表达明显 | 使用强化词 |
| 中等 | 情感表达适中 | 常规表达 |
| 弱 | 情感表达隐晦 | 使用委婉语、推测词 |

## 分析方法

### 基于规则的方法

#### 1. 情感词典匹配

**基本原理**：通过匹配文本中的情感词来判断情感倾向。

**实现步骤**：
```
1. 加载情感词典（正向词、负向词）
2. 对文本进行分词
3. 统计正向词和负向词数量
4. 计算情感得分
5. 根据得分判断情感倾向
```

**情感得分计算**：
```python
def calculate_sentiment_score(text):
    positive_words = ["好", "棒", "优秀", "出色", ...]
    negative_words = ["差", "弱", "不好", "糟糕", ...]

    positive_count = sum(1 for word in positive_words if word in text)
    negative_count = sum(1 for word in negative_words if word in text)

    score = positive_count - negative_count

    if score > 0:
        return "positive", score
    elif score < 0:
        return "negative", abs(score)
    else:
        return "neutral", 0
```

#### 2. 否定词处理

**问题**：否定词会翻转情感词的含义。

**示例**：
- "好用" → 正向
- "不好用" → 负向

**处理方法**：
```python
negation_words = ["不", "没", "非", "无", "不是", "没有"]

def handle_negation(text):
    words = segment(text)
    result = []

    i = 0
    while i < len(words):
        if words[i] in negation_words and i + 1 < len(words):
            # 下一个词的情感翻转
            next_word = words[i + 1]
            if next_word in positive_words:
                result.append(f"neg_{next_word}")  # 标记为负向
                i += 2
            elif next_word in negative_words:
                result.append(f"neg_neg_{next_word}")  # 负负得正
                i += 2
            else:
                result.append(words[i])
                i += 1
        else:
            result.append(words[i])
            i += 1

    return result
```

#### 3. 程度副词处理

**问题**：程度副词会增强或减弱情感强度。

**程度副词分类**：
| 类型 | 词汇 | 权重 |
|------|------|------|
| 极度 | 非常、特别、极其 | 2.0 |
| 高度 | 很、太、相当 | 1.5 |
| 中度 | 比较、稍微 | 1.2 |
| 低度 | 有点、些许 | 1.1 |

**处理方法**：
```python
degree_words = {
    "非常": 2.0,
    "特别": 2.0,
    "很": 1.5,
    "太": 1.5,
    "比较": 1.2,
    "稍微": 1.2,
    "有点": 1.1
}

def handle_degree(text):
    words = segment(text)
    scores = []

    i = 0
    while i < len(words):
        if words[i] in degree_words and i + 1 < len(words):
            next_word = words[i + 1]
            if next_word in positive_words:
                scores.append(degree_words[words[i]])
                i += 2
            elif next_word in negative_words:
                scores.append(-degree_words[words[i]])
                i += 2
            else:
                i += 1
        else:
            if words[i] in positive_words:
                scores.append(1.0)
            elif words[i] in negative_words:
                scores.append(-1.0)
            i += 1

    return sum(scores)
```

## 结果应用

### 舆情监控

**正向舆情应用**：
- 提取用户认可的优点

- 传播正面案例

**负向舆情应用**：
- 识别产品问题

**混合舆情应用**：
- 分析正负平衡点

### 产品改进

**问题优先级排序**：
```
高优先级：
- 负向情感强度高
- 影响用户数量多
- 出现频率高

中优先级：
- 负向情感强度中
- 影响用户数量中
- 出现频率中

低优先级：
- 负向情感强度低
- 影响用户数量少
- 出现频率低
```

### 用户运营

**用户分层**：
```
正面用户：
- 拥趸：深度合作
- 支持者：定期维护

负面用户：
- 批评者：了解诉求，争取转化
- 黑粉：区分对待，必要时公关

中性用户：
- 通过优质内容转化
- 通过活动激活
```

### 风险预警

**预警触发条件**：
1. 负向情感比例连续上升
2. 高强度负向情感出现
3. 特定问题集中爆发

**预警级别**：
| 级别 | 触发条件 | 响应时间 |
|------|---------|----------|
| 一级 | 负向比例>50%或高强度负面 | 1小时 |

### A. 情感分析代码示例

```python
class SentimentAnalyzer:
    """情感分析器"""

    def __init__(self):
        self.positive_words = self._load_positive_words()
        self.negative_words = self._load_negative_words()
        self.negation_words = ["不", "没", "非", "无", "不是", "没有"]
        self.degree_words = {
            "非常": 2.0, "特别": 2.0, "很": 1.5, "太": 1.5,
            "比较": 1.2, "稍微": 1.2, "有点": 1.1
        }

    def _load_positive_words(self):
        """加载正向词"""
        # 实际实现从文件加载
        return ["好", "棒", "优秀", "出色", "厉害", "强大", "满意", "推荐"]

    def _load_negative_words(self):
        """加载负向词"""
        # 实际实现从文件加载
        return ["差", "弱", "不好", "糟糕", "失望", "落后", "问题"]

    def analyze(self, text):
        """分析文本情感

        Args:
            text: 待分析文本

        Returns:
            dict: 情感分析结果
        """
        scores = []
        details = []

        words = list(text)  # 简化分词

        i = 0
        while i < len(words):
            # 检查程度副词
            degree = 1.0
            if words[i] in self.degree_words:
                degree = self.degree_words[words[i]]
                i += 1

            # 检查否定词
            negation = 1
            if i < len(words) and words[i] in self.negation_words:
                negation = -1
                i += 1

            # 检查情感词
            if i < len(words):
                word = words[i]
                if word in self.positive_words:
                    score = 1.0 * degree * negation
                    scores.append(score)
                    details.append({"word": word, "score": score})
                elif word in self.negative_words:
                    score = -1.0 * degree * negation
                    scores.append(score)
                    details.append({"word": word, "score": score})

            i += 1

        # 计算总得分
        total_score = sum(scores) if scores else 0

        # 判断情感倾向
        if total_score > 0:
            sentiment = "positive"
        elif total_score < 0:
            sentiment = "negative"
        else:
            sentiment = "neutral"

        return {
            "sentiment": sentiment,
            "score": total_score,
            "intensity": min(abs(total_score) / 3, 1.0),
            "details": details
        }

# 使用示例
analyzer = SentimentAnalyzer()
result = analyzer.analyze("文心大模型非常好用，我很满意")
print(result)
```

### B. 情感分析评估指标

| 指标 | 说明 | 计算方式 |
|------|------|----------|
| 准确率 (Accuracy) | 正确分类的比例 | (TP+TN)/(TP+TN+FP+FN) |
| 精确率 (Precision) | 正向预测的准确度 | TP/(TP+FP) |
| 召回率 (Recall) | 实际正向被识别的比例 | TP/(TP+FN) |
| F1分数 | 精确率和召回率的调和平均 | 2×P×R/(P+R) |
