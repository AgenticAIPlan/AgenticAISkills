# 能源行业大模型应用调研维度框架

## 三大核心调研维度

### 维度一：数据建设（Data Infrastructure）

调研重点：能源企业为大模型应用所做的数据基础设施建设。

**关键关注点：**
- 能源数据湖/数据中台建设
- 设备数据采集与标准化（SCADA、IoT传感器数据）
- 电力运行数据（调度数据、用电量数据、故障记录）
- 企业知识库构建（设备手册、运维规程、安全规范）
- 多模态数据建设（电力巡检图像、声纹、振动数据）
- 数据治理与数据质量
- 行业数据集开源或共建情况

**典型新闻特征词：**
`数据中台` `数据湖` `知识图谱` `数据治理` `工业数据` `能源数据集` `电力数据开放`

---

### 维度二：行业模型（Industry-Specific Models）

调研重点：专为能源行业训练或适配的大模型。

**关键关注点：**
- 能源行业专有大模型发布（预训练/微调）
- 主流大模型在能源场景的行业微调版本
- 电力行业垂直模型（如调度模型、预测模型）
- 多模态能源模型（图像+文本+时序）
- 模型评测基准（能源行业benchmark）
- 企业与科技公司联合研发模型的合作公告

**典型新闻特征词：**
`能源大模型` `电力大模型` `行业大模型` `垂直模型` `领域模型` `预训练` `微调` `行业benchmark` `联合研发`

**已知行业模型案例（参考）：**
- 国网"慧眼"（电力AI平台）
- 南方电网"能链大模型"
- 华为盘古能源大模型
- 远景能源"能碳大脑"

---

### 维度三：应用落地（Applications & Use Cases）

调研重点：大模型在能源生产、运营、服务各环节的具体落地应用。

**分类体系：**

| 应用场景 | 描述 | 典型关键词 |
|---------|------|----------|
| **智能运维** | 设备故障预测、缺陷检测、巡检机器人 | 预测性维护、智能巡检、缺陷识别 |
| **电力调度** | 负荷预测、新能源消纳、调度优化 | 负荷预测、调度智能化、新能源消纳 |
| **安全生产** | 安全风险识别、违规行为检测 | 安全帽、作业规范、风险预警 |
| **客服与营销** | 智能客服、电费咨询、用电分析 | 数字员工、智能坐席、客服机器人 |
| **知识管理** | 技术文档问答、规程检索、培训辅助 | 知识库、文档问答、技术规程 |
| **工程设计** | 电网规划、工程图纸辅助设计 | 辅助设计、电网规划、工程AI |
| **碳管理** | 碳排放核算、碳资产管理 | 碳中和、碳排放、碳资产、绿电 |
| **市场交易** | 电力现货市场、绿电交易辅助 | 电力市场、现货交易、绿证 |

## 调研周期建议

- **周报**：聚焦本周重大发布、合作公告
- **月报**：综合企业动态 + 主题分析 + 趋势研判
- **季报**：行业全景梳理 + 对标分析 + 深度洞察
