# IP小说改编短剧/漫剧剧本生产 SOP

帮助用户系统化地完成**小说改编短剧/漫剧剧本**的全流程创作，从拆书分析到剧本正文输出。

## 核心能力

### 📚 **Step 1：拆书分析**（深度理解原著，提取改编基石）

#### 1.1 通读小说
- 通读全篇小说或前 100 章（通常前 100 章为精华部分）
- 标记重要情节点、人物关系、名场面

#### 1.2 小说整体分析（5 大核心要素）
1. **小说梗概 + 核心卖点**
   - 用一段话概括故事主线
   - 用一句话提炼最抓人、最具市场吸引力的核心卖点
   - 示例：*女主作为豪门真千金被找回后，受尽假千金和家人的欺凌，最终含恨而死。重生后，她放弃讨好家人，转而凭借前世记忆疯狂搞事业，并联手幕后大佬对抗家族。核心卖点："重生归来，我让整个豪门跪地求饶"*

2. **情感引擎**：明确主导情绪（复仇、甜宠、逆袭、爽文、悬疑等）

3. **名场面清单**：列出原著中必保留的"高光时刻"和"经典台词"

4. **爽点提炼与分类**
   - 系统性地梳理和分类原著中的"爽点"（如打脸、逆袭、发糖、碾压、反转）
   - 规划爽点在不同剧集中的分布，确保节奏紧凑，持续吸引观众

5. **人物标记**
   - 提炼原著中主要/次要人物
   - 特别关注主要人物的情感转折点，以此标记关键情节点

#### 1.3 小说框架分析（4 大关注点）
1. **人物关系与背景**：构建角色互动的情感网络与世界观基底，明确故事发生的逻辑前提

2. **关键情节点梳理**：筛选驱动叙事的核心事件（如冲突/反转），串联改编剧本的骨架与节奏

3. **人物弧光/成长线**：人物在故事进程中，因经历事件、解决冲突产生的性格、认知、立场等方面的成长或转变
   - ⚠️ **人物弧光是改编基石**——若原著塑造出色，剧本则锦上添花；若原著弧光扁平，剧本需重构

4. **故事主题**：提炼原著核心立意与价值观（如成长、救赎、坚守等），决定改编后的情感共鸣点与市场定位

#### 1.4 确定集数与情节点
- **参考标准**：竖屏剧通常单集 1-2 分钟，总集数 60-80 集
- 针对小说多元素混杂的特性，结合竖屏剧市场规律，提取并深化核心亮点
- 根据情节密度决定最终集数

#### 1.5 人物设定及关系
- **主要人物**：提炼核心人设，关注主要人物弧光/成长线（如有标志性动作或台词可单独提出）
- **次要人物**：通常只沿用次要人物原著姓名，该人物数量需精简，避免过度挖掘
- ⚠️ **注意**：短剧中人物关系极简，次要人物须服务于主要人物，其核心功能为推动主要人物情感线发展

### 📝 **Step 2：剧本策划**（完成整体策划，搭建剧本框架）

#### 2.1 剧本名称 ✅
- 通常为一句话最抓人的卖点
- 示例：《重生归来，我让整个豪门跪地求饶》

#### 2.2 剧本集数 ✅
- 明确总集数（如 60 集、80 集）

#### 2.3 剧本类型 ✅
- 明确主导情绪定剧本类型
- 如：复仇、甜宠、逆袭、爽文等具体类型

#### 2.4 其他策划内容（可选）
- 【时代背景】通常年代剧必选
- 【核心冲突】【核心卖点】【禁忌】【付费卡点设计】等视编剧喜好选填

#### 2.5 故事梗概 ✅（6 大核心要素）
1. **开篇钩子**：用极具画面感的场景开场，立即抓住注意力
2. **主角介绍**：清晰描述主角的身份、处境和渴望
3. **核心冲突**：阐述主角面临的主要困境和对手
4. **金手指设定**：说明主角解决危机的独特能力
5. **故事走向**：概括主要剧情发展和转折点
6. **结尾悬念**：留下让人想继续看的悬念

#### 2.6 人物小传 ✅

**一般必含项**：
- **基本信息**：姓名、性别、年龄
- **身份背景**：社会地位、家庭出身、成长经历
- **核心事件**：打破人物平衡状态的关键冲突，确立其在剧中的核心任务

**选填项**（视角色核心设定或编剧风格选填）：
- **性格特点**：3-5 个核心标签
- **内心渴望**：表面目标 vs 深层需求
- **致命弱点**：性格缺陷或心理创伤
- **人物弧光**：在故事中的成长变化轨迹
- **人物关系**：与其他角色的关系网
- **经典台词**：3-5 句符合人物性格的标志性台词

**示例**：
> （基本信息）张婆：女，70 岁，（身份背景+形象）独居念旧的慈祥老人，（核心事件）因陈青山推进并村激烈反对，视老房为"老伴"的寄托。年轻时嫁给了村里的一位朴实憨厚的男人，两人相濡以沫，走过了无数个贫困的岁月。随着岁月流逝，丈夫因病去世，留下她一人守着老屋和一份家园的记忆。

#### 2.7 剧本集纲 ✅（6 大集纲要素）

**核心包含**：
- 每集故事走向及核心事件（如人物出场、情绪节奏、爽点设计、钩子设置）
- 通过集纲可以看出从一集开头至结尾的核心主线、亮点
- 可以参考起承转合，把故事拆分为四阶段来阐述

**集纲要素**：
1. **集数标题**：简洁有力的标题
2. **核心事件**：本集发生的主要剧情
3. **情绪节奏**：憋屈 → 憋屈+ → 爆发爽的层级设计
4. **爽点设计**：本集的情绪爆点（打脸、反转、高光）
5. **钩子设置**：结尾留下的悬念
6. **人物出场**：本集出现的主要角色

⚠️ **一卡/二卡/三卡结构说明**：
- **一卡（1-10 集）**：开篇破局，极速立人设、密集铺排爽点，利用强情绪钩子完成观众"入坑"转化，确立追看基础
- **二卡/三卡（11-60 集）**：矛盾层层递进，并在每一卡结尾设置强悬念三角形（付费卡点），利用"期待感"驱动用户持续追看及付费

### ✍️ **Step 3：剧本正文创作**（完成一卡/二卡/三卡剧本的具体内容）

#### 3.1 剧本格式规范 ✅

**基本信息**（每集必须标注）：
1. **集数**：每集开头标注第 X 集
2. **场号**：场景的序号（如 1-1 代表第一集第一个场景，1-2 代表第一集第二个场景）
   - 同一集中，更换不同场景，就得用不同的场号
   - 更换场景时需要空一行
3. **时间**：白天/晚上，在剧本中记为 日/夜
4. **环境**：内景/外景，在剧本中记为 内/外
5. **地点**：场景的具体地点（如办公室/茶水间等）
6. **人物**：这个场景中出现的主要人物

**剧本符号**：
- **△**：除台词外，能通过镜头演绎出来的部分（如人物动作、连贯的场景描写等）
- **( )**：括号一般伴随着人物台词，括号中的内容是指画面中人物的语气、表情、神态、情绪、动作等提示，并且一定伴随着人物讲这句台词时，同时发生的表情或动作
  - 示例：人名(冷笑)：台词内容 xxx
  - 如果不是同时发生的，而是做完动作以后再去说话的，则需用 △ 来代表演绎的部分
- **OS**：角色心理活动。记为：人名（OS）：心理台词内容
  - 注：OS 同时又有表情动作的，需先描述表情动作再描述 OS
  - 示例：柯南（勾唇 OS）：凶手就是那个人！
- **VO**：画外音。人物未出现，声音出现（如：接电话/广播里的声音）
- **闪回/闪出**：角色进入到回忆画面内容的标注

**格式示例**：
```
第 X 集：

场次 X：[日/夜 - 室内/室外 - 地点]
人物：[角色1]、[角色2]、[角色3]

△ [简洁的动作/场景描写，不超过2行]

角色名：（台词，需体现性格）
△ [伴随动作/反应]
```

**实际示例**：
```
第一集：

1-1：日 外 沈府院内
人物：沈枝熹、鸳鸯、大夫、侍女若干

△ 沈府，满院上下忙忙碌碌却各司其职，一阵风吹起院中樱花树上的风铃，花瓣落下，只见沈枝熹一身华服，躺在树下的贵妃椅上，旁边放着茶水和精致点心。

△ 鸳鸯立在沈枝熹身后，旁边一众侍女随侍，大夫紧张的在替沈枝熹把脉。

大夫：（擦着汗水，突然瘫倒在地）沈.....沈小姐......我知你还待字闺中，但.....这竟然是喜脉，您.....好像有孕了！

沈枝熹：什么！（站起来）

△ 大夫颤颤巍巍的跪下来。
```

#### 3.2 字数与时长标准
- 1-2 分钟/集 对应 500-800 字
- 2-3 分钟/集 对应 1000 字以内
- 按此标准，一部 80 集的短剧总剧本字数约为 **8 万字**

#### 3.3 创作建议
- 依据"一/二/三卡"剧本结构，覆盖第 1 集至 60 集（或更多）全量内容
- 遵循"集数-场次-人物-△xxx"格式推进创作，确保叙事连贯
- 每集结尾设置钩子/悬念，引导用户继续观看下一集

## 使用示例

### 示例 1：用户提供小说文本，要求改编剧本

**用户**：我有一个小说，讲的是女主重生复仇的故事，想改编成 60 集短剧，帮我生成剧本。

**助手**：
1. 先通读小说，提取核心卖点和爽点
2. 完成小说整体分析 → 框架分析 → 确定集数与情节点 → 人物设定
3. 生成剧本策划（名称、类型、梗概、人物小传、集纲）
4. 根据集纲创作一卡剧本（第 1-10 集）
5. 继续创作二卡/三卡剧本

### 示例 2：用户只提供故事概述，要求生成集纲

**用户**：我想做一个关于霸道总裁的短剧，女主是小助理，剧本类型是甜宠，帮我生成 80 集的集纲。

**助手**：
1. 根据用户提供的概述，补全核心冲突、金手指设定、爽点设计
2. 明确主导情绪（甜宠）
3. 生成 80 集的集纲，确保每集有爽点和钩子
4. 提供集纲预览，供用户确认

---

## 注意事项

1. **人物弧光是改编基石**：若原著人物弧光扁平，剧本需重构
2. **爽点分布要合理**：密集铺排爽点，确保节奏紧凑，持续吸引观众
3. **钩子设置要强**：每集结尾设置悬念，引导用户继续观看
4. **字数与时长要匹配**：1-2 分钟/集对应 500-800 字
5. **次要人物需精简**：短剧中人物关系极简，次要人物须服务于主要人物
