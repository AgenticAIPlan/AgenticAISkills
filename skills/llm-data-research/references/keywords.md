## 招募信号解读逻辑

通过招募内容反推模型发力方向：

| 招募信号 | 推断模型发力方向 |
|---------|----------------|
| 大量招募医学/临床专家标注 | 医疗健康垂直应用/对话能力 |
| 大量招募律师/法务人员标注 | 法律垂直应用/合规能力 |
| 大量招募代码/数学专家 | 代码生成/数学推理能力 |
| 大量招募多语言专家 | 国际化/多语言能力 |
| 大量招募视频理解标注 | 视频多模态能力 |
| 高单价RLHF对话标注 | Post-training/对话质量提升 |
| 合成数据质检任务 | 数据效率优化/减少人工标注依赖 |
| 招募有思维链能力标注专家 | 推理/思考型模型（CoT/reasoning） |

## 关键词扩展建议

每次调研时根据当前热点扩展关键词：
- 如有新模型发布（如 GPT-5、DeepSeek 新版），搜索 `{模型名} 数据标注 训练` 和 `{模型名} annotation recruitment`
- 如有重大融资事件，搜索 `{公司名} 融资 数据`
- 如有政策出台，搜索 `数据标注 政策 大模型`
- 如发现新的外部数商，立即跟踪其招募动态
