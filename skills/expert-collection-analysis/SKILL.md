# 智库专家数据采集与分析 Skill

## 业务场景

**智库专家信息管理与建联推荐**

在AI数据领域的发展过程中，需要持续跟踪权威专家（政府官员、智库学者、行业专家）的最新观点和言论，以便：
- 识别高价值专家进行建联合作
- 了解政策走向和行业趋势
- 筛选合适专家参与学术交流活动
- 建立AI数据领域的专家知识库

本Skill提供从**数据采集 → AI分析 → 优先级评分 → 专业导出**的完整解决方案。

## 核心功能

### 1. 权威信息采集 📡
- **15个权威信息源**：政府部门5个、权威智库5个、权威媒体5个
- **30个真实搜索URL**：每个信息源生成网页搜索和新闻搜索各1个URL
- **百度搜索集成**：使用baidu-search skill获取实时最新信息
- **时间过滤**：自动过滤24个月前的陈旧数据
- **智能去重**：按专家姓名+言论内容自动去重

**采集的信息源**：

| 类型 | 信息源 |
|------|--------|
| **政府部门** | 国务院、国家数据局、工业和信息化部、科技部、中央政府 |
| **权威智库** | 国务院发展研究中心、中国信息通信研究院、中国社会科学院、国家高端智库、清华大学 |
| **权威媒体** | 人民网、新华网、央广网、经济日报、财经网 |

### 2. AI智能分析 🤖
- **AI数据相关度评分**（0-100分）：基于发言内容与AI数据领域的相关性
- **权威性评分**（0-100分）：基于专家身份、机构地位、影响力
- **综合优先级判断**：
  ```
  综合分数 = 相关度 × 0.6 + 权威性 × 0.4

  高优先级 ⭐⭐⭐：综合分≥70 且 相关度≥60
  中优先级 ⭐⭐：综合分≥50 或 相关度≥40
  低优先级 ⭐：其他
  ```

### 3. 专家信息聚合 👥
- 按专家姓名聚合所有言论
- 补充专家档案信息（单位、职务、简介）
- 统计发言频率和活跃度
- 提取核心观点和关注领域

### 4. 专业数据导出 📄
- **Excel双工作表**：专家档案表 + 专家言论表
- **UTF-8 BOM编码**：完美支持中文显示，避免乱码
- **飞书多维表格兼容**：可直接导入飞书使用
- **结构化数据**：清晰的字段划分，便于二次分析

## 常见问题

**Q: 为什么有些链接无法直接打开文章？**
A: 生成的是百度搜索URL，需要通过浏览器访问后点击搜索结果查看原文。这样可以获取最新、最权威的信息。

**Q: 如何添加新的政府部门或智库？**
A: 编辑 `baidu_integration.py` 文件，在对应分类中添加新的搜索配置。

**Q: Excel显示乱码怎么办？**
A: 已使用UTF-8 with BOM编码，Excel应自动识别。如仍有问题，请确认使用的是Microsoft Excel或WPS。

**Q: 可以自定义优先级判断标准吗？**
A: 可以，修改 `ai_analyzer.py` 中的评分公式和阈值即可。

**Q: 采集失败会影响整体流程吗？**
A: 不会，系统会记录错误日志，继续处理其他信息源。
