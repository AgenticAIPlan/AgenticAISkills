# 组织结构规则

## 向上溯源

默认优先确认与用户目标直接相关的上级 / 主管 / 主办 / 出资关系，并满足单位性质判断、会前背景理解、合作判断、审批链条或资源协调判断所需的信息完整度。

只有在以下情况之一成立时，才继续上溯到更高层级，并允许进入最终报告图示：

- 用户明确要求查看完整上游链路。
- 不继续上溯就无法判断单位性质、治理归属、主管关系或出资关系。
- 该更高层级会直接影响本次沟通、审批、合作、汇报关系或资源协调判断。
- 对政府机构、事业单位、国有企业、社会组织等，最高主管机关或最高出资 / 管理层级是判断其权限边界、资源动员能力或合作路径的必要信息。

如果更高层级仅属于泛背景信息，且对当前任务没有直接帮助，则默认不写入最终报告图示，可保留在候选来源、原子事实、组织分析或待核实材料中。

按需上溯参考：

- 党政机关：先确认直接上级或管理机关；若最高国家机关会影响单位性质、权限边界、审批链条或合作判断，可继续上溯并说明业务相关性。
- 人民团体 / 群团组织：先确认直接主管、主办或所属关系；若最高组织领导关系会影响会前沟通、资源协调或合作判断，可继续上溯并说明业务相关性。
- 事业单位：先确认主管部门 / 主办单位 / 所属机关；若主管机关的更高层级会影响单位性质、资源协调或合作路径判断，可继续上溯并说明业务相关性。
- 国有企业：先确认集团公司、履行出资人职责机构、国资委或财政部门；若继续穿透到最高可确认国有出资层级有助于判断决策链条或合作路径，可继续上溯并说明业务相关性。
- 民营企业：上溯到最终控股股东、实际控制人、最顶层投资人或自然人；无法继续穿透时，停在最高可确认企业或自然人。
- 社会组织：登记机关、业务主管单位、党建工作机构按可确认来源分别列示；若业务主管单位的更高层级对会前沟通、资源协调或合作判断有直接帮助，可继续上溯并说明业务相关性。

隶属关系图示不得只画一个直接上级，除非已经说明继续上溯的检索路径、不可确认原因，或确认更高层级对当前任务无直接业务必要性。

## 图示列全规则

- 对已经由官网或权威来源确认完整的清单，`隶属关系图示` 中不得用“等”省略。
- 需要列全的清单包括：内设机构、直属单位、投资兴办企业、控股参股企业、主管社会组织、标委会、学会协会、联合实验室、工作组等。
- 如果来源只支持部分确认，图示中必须写明“部分确认”，并在完整性声明中说明缺口和已检索路径。
- 若清单过长，仍应列全；可以按类型分组，但不得用“等”替代已确认名称。

## 向下梳理

覆盖：

- 内设机构。
- 直属事业单位。
- 控股或参股企业。
- 主管社会组织。
- 业务指导 / 业务主管 / 技术归口的组织。
- 标准化技术委员会、专家委员会、咨询委员会、评审委员会。
- 秘书处承担单位、工作组、专委会、联盟、联合实验室、业务协同平台。
- 其他由官网、主管部门、国家标准委、登记平台、公告或制度文件确认的业务关联组织。

## 非行政隶属关联

遇到“业务指导单位”“业务主管单位”“技术归口”“指导单位”“主办单位”“秘书处承担单位”“承办单位”“依托单位”“工作组”“专家委员会”“标准化技术委员会”等表述时：

- 必须作为候选来源和原子事实提取，不得只因其不是直属单位而忽略。
- 在 `隶属关系图示` 中可单列为 `业务指导 / 技术归口 / 专家咨询 / 标准化` 等分支，使用括注标明关系类型。
- 在 `下属及关联单位` 中按关系类型归类说明。
- 不得写成行政下属、直属单位或控股单位；若关系类型不是行政隶属，必须显式写明“非行政隶属”。
- 如果只从新闻活动确认“参与 / 出席 / 协同”，不能写成稳定关联；只能写入待核实线索或近期动态。

示例：

```text
国家数据局
├── 内设机构：综合司、政策和规划司...
└── 非行政隶属关联
    └── 全国数据标准化技术委员会（业务指导单位：国家数据局；秘书处：中国电子技术标准化研究院）
```

## 完整性声明

每类关系必须声明完整性：

- `完整`：官网或权威页面提供完整清单。
- `部分确认`：只从新闻、招聘、招标、年报中确认部分名称。
- `无法确认`：未找到可核查来源。

不得把部分确认写成完整清单。

## 图片机构图 Fallback

- 若官网只以图片展示机构设置、组织架构或职能图，且无法直接提取文字，不得当场放弃。
- 必须保存截图或在报告中写明截图路径；如果无法保存截图，必须说明无法保存的原因。
- 同时继续通过至少 2 种间接渠道补全文字拼图：上级主管部门页面、招聘/招标公告、新闻通稿、预算决算、事业单位登记、工商或社会组织平台。
- 若仍无法补全，内设机构完整性必须标为 `无法确认` 或 `部分确认`，不得写成完整清单。

## 常见误区

- 展厅、业务平台、栏目、项目不是行政内设机构。
- 友情链接不是隶属关系。
- 共建单位不是下属单位。
- 试点单位不是主管或被主管单位。
- 标委会、专家委员会、联盟、工作组等不是当然的直属下属；必须标明“业务指导”“技术归口”“秘书处承担”等具体关系。
