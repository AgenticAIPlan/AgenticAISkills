# 国家数据局 org-research 高质量报告样例（节选版）

本样例基于前期真实测试案例整理，用于展示 `org-research` 的推荐输出形态：先证据、再准入、后成稿，并在正文后保留可复核附录。

为避免 reference 文件过大导致评审脚本截断，本文件只保留代表性章节和代表性表格行。正式交付时应按 `assets/report-template.md` 保留完整附录。

## 1. 调研报告正文

### 1.1 报告状态

正式报告。核心事实均来自公开网页，并经过候选来源、原子事实、链接审计、事实准入和 Reviewer 终审。

### 1.2 单位基本介绍

国家数据局是国务院直属机构，由国家发展和改革委员会管理。其设立和职责配置可从国务院机构改革方案、国务院任免信息、国家数据局官网及相关公开页面中交叉核验。[国务院关于机构设置的通知](https://www.gov.cn/zhengce/content/2023-03/20/content_5747443.htm) [国家数据局官网](https://www.nda.gov.cn/)

### 1.3 主要职责

- **数据基础制度建设**：统筹推进数据基础制度建设，协调推进数据资源整合共享和开发利用。[国务院机构改革方案](https://www.gov.cn/xinwen/2023-03/11/content_5745977.htm)
- **数字中国建设统筹**：统筹推进数字中国、数字经济、数字社会规划和建设。[国务院机构改革方案](https://www.gov.cn/xinwen/2023-03/11/content_5745977.htm)
- **数据要素市场建设**：推动数据要素市场建设和数据资源开发利用相关工作。[国家数据局官网](https://www.nda.gov.cn/)

### 1.4 隶属关系图示

```text
国务院
└── 国家发展和改革委员会
    └── 国家数据局
```

业务相关性说明：国家数据局由国家发展和改革委员会管理。上溯至国务院和国家发展改革委，有助于判断其制度制定、跨部门协调、资源配置和会前沟通链条，因此进入图示。

### 1.5 内设机构

公开信息可确认国家数据局官网设置了“机构”栏目，但完整内设司局清单需以官网最新披露为准。若官网未披露完整清单，报告应标注“部分确认”或“无法确认”，不得用新闻、活动栏目或第三方介绍补全。

### 1.6 主要负责人

- 刘烈宏：公开信息显示其曾任国家数据局党组书记、局长。该身份应以国务院任免、国家数据局官网或人民网等权威公开来源交叉核验。[国务院任免国家工作人员](https://www.gov.cn/gwyzzjg/renmian/)
- 沈竹林：公开信息显示其曾任国家数据局副局长。具体分工和履历应以国家数据局官网“局领导”栏目或权威任免信息为准。[国家数据局官网](https://www.nda.gov.cn/)

缺口说明：负责人分工、完整履历和最新领导班子应以官网最新页面为准。若无法访问或页面未披露，不得拼凑成完整领导班子。

### 1.7 近期动态

- 2026-04-未查明日期：国家数据局围绕数据要素、数字中国建设等方向发布或转载相关政策动态。正式报告应逐条链接到具体新闻或政策页面，不得使用栏目页替代。[国家数据局官网](https://www.nda.gov.cn/)
- 2026-04-未查明日期：如涉及“可信数据空间”“公共数据资源开发利用”等主题，应回到具体通知、新闻或政策原文核验后再写入正文。

## 2. 待核实线索

- 官网如未完整公开内设机构，不应补全为完整架构。
- 负责人最新分工如无公开页面支撑，应列为缺口。
- 仅在新闻中出现的项目、平台、工作组、委员会，不得直接写成现行内设机构或直属单位。
