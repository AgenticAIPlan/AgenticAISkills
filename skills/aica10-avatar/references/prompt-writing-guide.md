# ERNIE-Image 提示词写作指南

## 核心结构

```
主体 + 场景 + 风格 + 色彩/光影 + 构图 + 氛围 + 质感 + 文字（如有）
```

## 关键原则

- **具体胜于抽象**：用"Kodak Portra 400 胶片颗粒感"代替"复古感"
- **明确命名风格**：日系动漫、胶片摄影、国风水墨、赛博朋克等
- **中英文混用**：精准术语可用英文，如 `bokeh`、`golden hour`、`cinematic lighting`

## 提示词长度参考

| 类型 | 字数 | 适用场景 |
|------|------|---------|
| 精准复现版 | 150–300 字 | 最大还原度 |
| 创意改编版 | 80–150 字 | 风格保留，灵活发挥 |
| 精简核心版 | 30–80 字 | 快速生成，验证风格 |

## 文字渲染建议

- 图中文字控制在 4–8 个汉字效果最佳
- 需显式说明文字位置和风格（如"底部居中，粗体白字"）

## 常见误区

| 错误写法 | 正确写法 |
|---------|---------|
| "没有云" | "晴空万里，蔚蓝天空" |
| 关键词堆砌 | 保持自然句式流畅 |
| "好看的" "漂亮的" | 具体描述光影、色调、风格 |

## 头像场景专项建议

生成用于社交头像的照片时，建议在提示词中加入：

- `正面人像，面部清晰`
- `简洁背景` 或指定背景色
- `500x500 正方形构图`
- 光线：`柔和自然光` 或 `室内暖光`
