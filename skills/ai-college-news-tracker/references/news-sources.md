# 推荐新闻来源列表

> 本文档整理了高校人工智能学院新闻的主要获取渠道。

## 高校官网/学院官网

### 获取方式
- 直接访问学院官网"新闻动态"、"学院公告"栏目
- 关注学院官方微信公众号
- 订阅学院RSS订阅源（如有）

### 注意事项
- 官网新闻为第一手资料，可信度最高
- 需要定期访问或设置更新提醒
- 部分学院官网更新频率较低

## 搜索引擎

### 搜索技巧

**关键词组合**
```
"人工智能学院" + site:edu.cn
"人工智能研究院" + site:edu.cn
"人工智能学院" + "成立"
"人工智能学院" + "合作"
```

**时间过滤**
- Google：使用"工具"->"任何时间"选择时间范围
- 百度：使用搜索工具设置时间范围

**高级搜索**
```
site:edu.cn "人工智能学院" 新闻
inurl:ai "学院" 新闻
```

## 使用建议

1. **优先级排序**：官网 > 新闻媒体 > 社交媒体 > 搜索引擎
2. **交叉验证**：重要新闻需多方验证
3. **时效把控**：定期检查，避免遗漏重要信息
4. **信息存档**：建立本地新闻库，便于回溯分析
