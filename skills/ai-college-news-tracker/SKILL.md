# AI College News Tracker

## 适用场景

当用户需要了解全国本科以上高校人工智能学院的最新动态、新闻资讯时，使用本 Skill。适用于：

- 了解人工智能学院的建设进展
- 追踪高校AI领域的重大新闻
- 收集人工智能教育相关的政策动态
- 获取高校AI研究成果和合作信息

## 输入要求

- **查询范围**（可选）：指定关注的高校、地区或特定主题
- **时间范围**（可选）：如"最近一周"、"最近一个月"
- **关键词过滤**（可选）：如"合作"、"招聘"、"研究"等
- **输出详细程度**：摘要版或详细版

## 执行步骤

### 第一步：确定高校人工智能学院清单

1. 搜索并整理全国本科以上高校中已设立人工智能学院的院校名单
2. 重点关注：
   - 独立设置的人工智能学院
   - 人工智能研究院/研究所
   - 以AI为核心的教学科研机构
3. 建立学院官网链接清单，存入 `references/ai-colleges-list.md`

### 第二步：多渠道新闻收集

按以下优先级收集新闻：

1. **高校官网/学院官网**
   - 访问学院官网新闻栏目
   - 关注学院公告、学术动态、合作新闻

2. **新闻媒体**
   - 科技类媒体：36氪、机器之心、AI科技评论等
   - 教育类媒体：中国教育报、科学网等
   - 综合类媒体：新华网、人民网教育频道

3. **社交媒体**
   - 微信公众号：高校官方号、学院公众号
   - 微博：高校官微、学院官微
   - 知乎：相关话题讨论

4. **搜索引擎**
   - 使用关键词组合搜索："人工智能学院 + 高校名称"
   - 设置时间过滤获取最新结果

### 第三步：新闻筛选与分类

1. 按新闻类型分类：
   - 学院建设（成立、更名、合并）
   - 师资动态（引进、任命、获奖）
   - 学术研究（论文、项目、成果）
   - 产学研合作（企业合作、实习基地）
   - 招生培养（专业设置、招生信息）
   - 会议活动（学术会议、论坛讲座）

2. 按重要性排序：
   - 高：重大政策、学院成立、重要合作
   - 中：学术成果、师资引进、重要活动
   - 低：日常通知、一般活动

### 第四步：生成结构化报告

按以下格式输出 Markdown 文档：

```markdown
# 高校人工智能学院新闻动态报告

## 报告概要
- 查询时间：YYYY-MM-DD
- 时间范围：YYYY-MM-DD 至 YYYY-MM-DD
- 覆盖高校数量：XX 所
- 新闻总数：XX 条

## 重点新闻

### [新闻标题]
- 来源：[高校名称]人工智能学院
- 时间：YYYY-MM-DD
- 类型：[学院建设/师资动态/学术研究/产学研合作/招生培养/会议活动]
- 摘要：[新闻摘要，100字以内]
- 链接：[原文链接]

## 分类汇总

### 学院建设
...

### 师资动态
...

### 学术研究
...

### 产学研合作
...

### 招生培养
...

### 会议活动
...

## 高校人工智能学院清单

| 高校名称 | 学院名称 | 官网链接 | 最新更新 |
|---------|---------|---------|---------|
| ... | ... | ... | ... |

## 待关注事项
- [需要持续关注的事件或趋势]
```

## 输出要求

- **格式**：Markdown 文档
- **结构清晰**：按分类组织，便于快速定位
- **信息完整**：每条新闻包含来源、时间、类型、摘要、链接
- **时效性**：优先展示最近时间段的新闻
- **可追溯**：保留原文链接，便于深入了解
- **客观中立**：如实呈现新闻内容，不做主观评价
