# 图片逆向分析提示词模板

以下是调用 ERNIE-4.5 多模态 API 时，作为 `text` 字段发送的分析提示词。

## 核心分析提示词

```
你是一位专业的视觉分析师和文生图提示词工程师，擅长从图片中逆向推导出精确的文生图提示词。

请对这张图片进行全方位深度分析，按以下 8 个维度逐一拆解：

## 1. 主体内容
- 画面中的核心对象是什么？（人物/动物/建筑/风景/物品等）
- 如果有人物：描述发型、发色、服装（材质/颜色/款式）、表情、动作/姿态、体型、年龄段
- 如果有场景：描述场景类型（室内/室外/幻想空间）、具体场景元素（家具/植物/天空/建筑等）
- 前景、中景、背景各有什么元素？

## 2. 艺术风格
从以下类型中判断最匹配的风格（可以是多种混合）：
- 日系动漫 / 美式卡通 / 国风水墨 / 欧美写实 / 赛博朋克 / 蒸汽朋克
- 胶片摄影 / 数码摄影 / 宝丽来 / 电影画面
- 水彩 / 油画 / 素描 / 像素艺术 / 矢量插画 / 扁平设计
- 超现实 / 极简主义 / 波普艺术 / 哥特风 / 洛可可
请具体描述风格特征，不只是给出名称。

## 3. 构图与视角
- 视角：俯拍/仰拍/平视/鸟瞰/鱼眼/特写/远景/中景？
- 构图法则：三分法/对称构图/对角线/引导线/框架构图/中心构图？
- 景深：全景深/浅景深（前景虚化/背景虚化）？
- 画面比例和裁切方式

## 4. 色彩分析
- 主色调（列出 2-3 个主要颜色及其大致 hex 值）
- 配色方案：互补色/类似色/单色/三色/分裂互补？
- 色温：冷色调/暖色调/中性？
- 饱和度：高饱和/低饱和/局部高饱和？
- 对比度：高对比/低对比/局部对比？

## 5. 光影效果
- 光源方向：正面光/侧光/逆光/顶光/底光？
- 光线类型：自然光/人工光/霓虹灯光/烛光/月光？
- 光线质感：硬光（锐利阴影）/柔光（渐变阴影）？
- 特殊光效：丁达尔效应/光晕/镜头光斑/体积光？

## 6. 氛围与情绪
- 整体氛围关键词（如：温馨/忧郁/热血/宁静/诡异/浪漫/史诗/日常）
- 情绪表达方式（通过色彩/光影/构图如何传达情绪？）

## 7. 文字内容（重点分析）
- 图中是否有文字？如果有：
  - 文字的具体内容是什么？（逐字记录）
  - 字体风格：衬线体/无衬线/手写体/像素字/哥特体/艺术字？
  - 文字颜色和大小
  - 排版位置和方式（居中/角落/沿曲线/融入画面）
  - 文字特效：发光/阴影/描边/渐变/3D 立体/故障效果？
  - 文字与画面的关系（标题/装饰/叙事元素/水印）

## 8. 纹理与特殊效果
- 画面整体质感：光滑/颗粒感/噪点/磨砂？
- 是否有胶片颗粒/老照片做旧效果？
- 特殊效果：烟雾/雨滴/雪花/花瓣/光粒子/故障艺术（glitch）？
- 边框和装饰效果

请用结构化的方式输出分析结果，每个维度单独成段，尽可能具体和精确。
如果某个维度在图片中不明显或不适用，简要说明即可，不要编造不存在的内容。
```

## 文字渲染增强提示词

当检测到图片中包含文字元素时，追加以下分析：

```
请特别关注图中的文字渲染部分，进行额外的深度分析：

1. **精确文字内容**：逐字逐行记录所有可见文字，包括标点符号
2. **字体识别**：尽可能识别具体字体名称或最接近的字体类型
3. **排版细节**：字间距、行间距、对齐方式、文字大小层级
4. **文字与背景的关系**：透明度、叠加模式、文字是否有背景底色
5. **文字动效暗示**：如果文字看起来有动态效果（如扭曲、弹跳、飘动），描述这种效果

这些信息对于在文生图模型中复现文字渲染效果至关重要。
```

## 动漫风格增强提示词

当图片为动漫风格时，追加以下分析：

```
这是一张动漫/二次元风格的图片，请额外关注：

1. **画风流派**：更接近哪位画师/哪部作品的风格？（如新海诚风格、京都动画风格、Makoto Shinkai style 等）
2. **线条特征**：线条粗细、是否有描边、线条颜色（黑线/彩色线/无线稿）
3. **眼睛画法**：眼睛大小、形状、高光位置、瞳孔细节
4. **头发渲染**：发丝细节、高光块面、阴影层次
5. **赛璐璐 vs 厚涂**：是传统赛璐璐风格还是厚涂混合？
```

## 胶片风格增强提示词

当图片为胶片/复古摄影风格时，追加以下分析：

```
这张图片具有胶片/复古摄影风格，请额外关注：

1. **胶片类型猜测**：更接近哪种胶片效果？（如 Kodak Portra 400、Fuji C200、Kodak Gold 200、CineStill 800T 等）
2. **颗粒感**：颗粒大小和密度
3. **色彩偏移**：高光偏色（偏暖/偏绿）、阴影偏色（偏蓝/偏紫）
4. **暗角和漏光**：是否有暗角效果或漏光（light leak）
5. **锐度特征**：整体锐度、边缘模糊
6. **年代感**：更接近哪个年代的摄影风格（60s/70s/80s/90s/Y2K）
```
