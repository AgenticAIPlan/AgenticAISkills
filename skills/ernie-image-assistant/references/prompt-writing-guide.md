# ERNIE-Image 提示词撰写指南

## 基本规范

ERNIE-Image 接受**中文自然语言描述**作为提示词，以下是撰写高质量提示词的核心原则。

### 提示词结构（推荐顺序）

```
[主体描述] + [场景/背景] + [艺术风格] + [色彩/光影] + [构图/视角] + [氛围/情绪] + [质感/特效] + [文字内容（如有）]
```

### 核心原则

1. **具体胜于抽象** — "穿着白色连衣裙的少女站在向日葵花田中" 优于 "一个女孩在花丛里"
2. **视觉词汇优先** — 使用可以直接转化为视觉元素的描述词
3. **风格明确标注** — 明确指出画风（如"日系动漫风格"、"胶片摄影风格"）
4. **适当使用中英混合** — 部分艺术概念用英文更精准（如 "bokeh"、"golden hour"、"film grain"）

## 各维度撰写要点

### 主体描述
- 人物：性别 + 年龄段 + 发型发色 + 服装详细描述 + 动作姿态 + 表情
- 场景：场景类型 + 具体元素列举 + 空间关系
- 物品：材质 + 颜色 + 形状 + 状态

**示例：**
- 好：`一位短发少女，黑色齐耳短发，穿着米白色oversized毛衣，双手捧着一杯热可可，微微低头，嘴角带着浅笑`
- 差：`一个女生拿着杯子`

### 风格描述
常用的风格关键词：

| 风格类型 | 关键词示例 |
|---------|-----------|
| 日系动漫 | 日系动漫风格、赛璐璐渲染、新海诚风格天空、京都动画风格 |
| 胶片摄影 | 胶片质感、Kodak Portra色调、富士胶片色彩、复古胶片颗粒感 |
| 写实摄影 | 高清摄影、专业棚拍、自然光人像、街头纪实摄影 |
| 水彩/插画 | 水彩渲染、淡彩插画、手绘质感、水彩晕染效果 |
| 电影画面 | 电影感画面、宽银幕比例、cinematic lighting、电影调色 |
| 赛博朋克 | 赛博朋克风格、霓虹灯光效、科幻都市、cyberpunk neon glow |
| 国风 | 中国风水墨画、工笔画风格、敦煌壁画风格、古风仙侠 |

### 色彩与光影
- 明确色温："暖色调的夕阳余晖" / "冷蓝色的月光氛围"
- 描述光源："金色侧光照亮面部" / "逆光剪影效果"
- 特殊光效："丁达尔效应光束" / "霓虹灯反射在雨水地面上"

### 构图与视角
- 视角："特写镜头" / "全身立绘" / "俯瞰视角" / "低角度仰拍"
- 构图："中心对称构图" / "三分法构图" / "大面积留白"
- 景深："背景虚化（大光圈效果）" / "浅景深梦幻感"

### 文字渲染（重点）
ERNIE-Image 支持在图中渲染文字，但需要精确描述：

- **明确标注文字内容**：`画面上方有"春日物语"四个字`
- **描述字体风格**：`手写体文字` / `粗体标题字` / `像素风格文字`
- **标注位置**：`左上角有文字` / `底部居中排列`
- **文字特效**：`发光文字效果` / `文字带有投影` / `渐变色文字`

**文字渲染示例：**
```
画面中央偏上位置有"夏日回忆"四个白色手写体文字，文字带有柔和的光晕效果，
文字下方是一片薰衣草花田，夕阳西下，胶片摄影风格，暖色调
```

### 质感与特效
- 胶片质感："胶片颗粒感、轻微过曝、色彩偏移"
- 梦幻效果："柔焦、光斑散景（bokeh）、朦胧感"
- 复古做旧："老照片质感、褪色效果、暗角"

## 提示词长度建议

| 版本 | 长度 | 用途 |
|------|------|------|
| 精准复现版 | 150-300 字 | 高度还原所有细节 |
| 创意改编版 | 80-150 字 | 保留核心风格，留出创意空间 |
| 精简核心版 | 30-80 字 | 快速出图，抓取画面精髓 |

## 常见误区

1. **避免否定描述** — "没有云的天空" 不如 "万里无云的湛蓝天空"
2. **避免过度堆砌** — 关键词堆砌会降低生成质量，保持语义通顺
3. **避免抽象概念** — "孤独感" 不如 "独自一人坐在空旷的公园长椅上"
4. **注意文字数量** — 图中渲染文字不宜过多，4-8 个汉字效果最佳
