# 专家运营内容采集与策略沉淀 Playbook

## 标准流水线

1. 梳理目标与产物
2. 用飞书多维表格建立源表、拆解表、整合表
3. 搜索近 3 年高相关内容
4. 写入内容源采集库
5. 按统一口径打分、分层
6. 过滤低价值内容
7. 从高分文章中提取策略卡片
8. 对策略卡片做去重整合
9. 生成策略笔记
10. 发布为飞书文档或继续回写表格

## 适用阶段标签建议

- 专家招募
- 专家冷启动
- 内容供给
- 分层运营
- 激励机制
- 活跃与留存
- 商业化转化
- 数据分析
- 综合策略

## 策略卡片最小结构

```markdown
## 策略名称
- 适用阶段：
- 核心打法：
- 执行步骤：
- 适用前提：
- 风险点：
- 参考分值/分层：
- 来源文章：
```

## 整合库命名建议

整合后的母策略名称要体现“动作 + 场景 + 价值”，不要直接复用文章标题，也不要只写抽象词。

好例子：

- 种子专家首周激活与社群冷启动SOP
- 专家栏目化内容生产、分发与复盘SOP
- 成长值-勋章-等级体系驱动的长期贡献激励机制

坏例子：

- 社群方法论
- 运营经验总结
- 高质量内容策略
