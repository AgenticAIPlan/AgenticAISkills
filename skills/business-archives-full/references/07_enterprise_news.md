# 板块七：企业新闻提示词

## 提示词模板

```
以这种格式总结一下{公司名称}的新闻，不要表格！，最好是有和其他企业合作方面的新闻

【参考格式示例】
1. 2024.12.23【艾普工华在重型商用车智造领域斩获新订单】：艾普工华在重型商用车智造领域斩获新订单...
2. 2024.07.19【艾普工华受邀参加湖北省工业软件产业创新联盟第一届理事会】：...
3. 2024.03.15【艾普工华与XX企业签署战略合作协议】：双方将在智能制造领域展开深度合作...
4. 2023.11.08【艾普工华荣获"专精特新"企业认定】：...

企业新闻应包含：
- 新闻日期
- 新闻标题（简明扼要）
- 新闻摘要（1-2句话）
```

## 使用说明

1. 将 `{公司名称}` 替换为实际公司名称
2. 发送完整提示词给 AI
3. 复制生成内容到档案"企业新闻"栏目
4. 按时间倒序排列（最新的在前）

## 注意事项

- 优先选择企业合作、战略签约、新产品发布等新闻
- 避免选择仅有宣传性质的无实质内容新闻
- 时间格式统一为"YYYY.MM.DD"
- 建议收集最近1-2年的新闻
