# 板块二：创始人信息提示词

## 提示词模板

```
以这种格式为模版输出{公司名称}创始人的信息，不要表格！！

【参考格式示例】
黄刚，艾普工华科技（武汉）有限公司董事长，2004年毕业于华中科技大学，获得工业与制造系统工程专业博士学位。曾任华中科技大学副研究员、德国Fraunhofer IPT研究所研究员。长期从事智能制造、工业软件研究，主持过多项国家科技重大专项...

创始人应包含：
- 姓名和职务
- 教育背景
- 职业经历时间线
- 主要成就/荣誉
```

## 使用说明

1. 将 `{公司名称}` 替换为实际公司名称
2. 发送完整提示词给 AI
3. 复制生成内容到档案"创始人信息"栏目
4. 格式为段落文本，非表格

## 注意事项

- 如果公司有多位创始人，选择最具代表性的1-2位
- 重点突出与当前业务相关的背景
