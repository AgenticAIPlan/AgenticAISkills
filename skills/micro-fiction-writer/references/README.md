# Micro Fiction Writer - References

本目录包含微小说创作的参考资料和示例。

## 目录结构

```
references/
├── README.md           # 本文件
├── healing-examples/   # 治愈系微小说示例
├── sarcastic-examples/ # 犀利系微小说示例
└── release-examples/   # 释怀系微小说示例
```

## 文风分类说明

### 治愈系 (Healing)
- **核心公式**：困境 → 微小善意/自我和解 → 温暖反转
- **适用场景**：孤独、疲惫、需要被理解的情绪
- **关键词**：室友留灯、陌生人的善意、被遗忘的细节

### 犀利系 (Sarcastic)
- **核心公式**：荒诞困境 → 黑色幽默 → 真相刺痛
- **适用场景**：职场挫折、社会对比、存在焦虑
- **关键词**：朋友圈对比、房租涨价、社畜日常

### 释怀系 (Release)
- **核心公式**：执念/遗憾 → 时间跨度 → 顿悟/和解
- **适用场景**：前任、怀旧、未完成的告别
- **关键词**：旧物、三年前的信、终于懂了

## 扩展规划

如需添加新的文风类型，请在本目录创建对应的示例文件夹：

```
references/
├── suspense-examples/  # 悬疑系（待扩展）
├── romantic-examples/  # 浪漫系（待扩展）
└── ...
```

## 使用说明

这些参考资料用于：
1. 展示不同文风的具体写法
2. 提供情节模式和反转技巧的参考
3. 作为评估模型输出质量的对比基准
