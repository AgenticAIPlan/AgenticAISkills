# 高价值 PR 关键词列表

本文档列出用于识别高价值 PR 的关键词分类。

## 新模型支持

关键词：支持新模型、新增模型、model support, add model, support new model

示例：
- "support Qwen3"
- "add Gemma support"
- "support new architecture"

## 新推理能力

关键词：新推理能力、新特性、新功能、推理能力、inference capability

示例：
- "add continuous batching"
- "support speculative decoding"
- "new quantization method"

## 性能优化

关键词：性能优化、提升速度、加速、performance, optimize, speed up, benchmark

示例：
- "improve throughput 2x"
- "reduce latency"
- "optimize memory usage"

## 重大功能更新

关键词：重大更新、核心功能、major feature, core functionality

示例：
- "redesign scheduler"
- "new API endpoint"
- "breaking change"

## Bug 修复

关键词：修复、bug fix、hotfix、critical fix

示例：
- "fix memory leak"
- "resolve race condition"
- "fix incorrect output"

## 可宣传性评级

| 评级 | 标准 |
|------|------|
| ⭐⭐⭐⭐⭐ | 新模型支持 + 知名模型 + 完整实现 |
| ⭐⭐⭐⭐ | 新推理能力或重大性能提升 |
| ⭐⭐⭐ | 功能优化或中等复杂度修复 |
| ⭐⭐ | 常规 Bug 修复或小优化 |
| ⭐ | 文档更新、配置修改等 |
