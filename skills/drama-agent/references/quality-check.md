# 质量检查详细说明

## 一致性检查

### 人物一致性

**检查要点**：
- 性格是否前后一致
- 说话方式是否一致
- 行为逻辑是否合理
- 动机是否清晰且稳定

**常见问题**：
- 突然性格大变（无铺垫）
- 说话方式突然改变
- 做出违背人设的行为

**示例检查**：
```
❌ 问题示例：
第1集：李明性格温和，不轻易发怒
第10集：李明突然暴怒，大吼大叫（无原因）

✅ 修正方案：
铺垫：李明长期压抑，某个事件触发爆发
或：调整第1集人设，暗示他有压抑的一面
```

### 时间线一致性

**检查要点**：
- 时间顺序是否合理
- 跨集时间衔接是否自然
- 事件发生时间是否有矛盾

**常见问题**：
- 时间跳跃无说明
- 事件时间线矛盾
- 人物年龄/时间计算错误

**示例检查**：
```
❌ 问题示例：
第5集：李明说"我30岁了"
第15集：李明说"我今年35岁"（只过了10集，不可能长5岁）

✅ 修正方案：
统一年龄或设定时间跨度
```

### 关系一致性

**检查要点**：
- 人物关系是否清晰
- 关系变化是否有逻辑
- 隐藏关系是否合理揭露

**常见问题**：
- 关系突然改变（无铺垫）
- 关系矛盾（前后说法不一致）
- 隐藏关系揭露突兀

**示例检查**：
```
❌ 问题示例：
第1集：李明和张伟是多年好友
第10集：突然揭露李明一直恨张伟（无铺垫）

✅ 修正方案：
在早期埋下伏笔：
- 第3集：李明看张伟的眼神闪烁
- 第6集：李明拒绝张伟的帮助
- 第8集：暗示两人有过去
```

### 能力/物品一致性

**检查要点**：
- 人物能力是否合理
- 物品获取是否有逻辑
- 能力/物品使用是否一致

**常见问题**：
- 突然获得新能力（无解释）
- 物品凭空出现
- 能力忽强忽弱

**示例检查**：
```
❌ 问题示例：
第10集：李明突然会说英语（之前从未提过）

✅ 修正方案：
提前铺垫：
- 第5集：李明在偷偷学英语
- 第8集：李明用英语看新闻被撞见
```

## 逻辑合理性检查

### 情节逻辑

**检查要点**：
- 事件发生是否有因果关系
- 人物行为是否有动机
- 结果是否合理

**常见问题**：
- 事件突然发生（无原因）
- 人物行为无动机
- 结果不符合常理

**示例检查**：
```
❌ 问题示例：
李明突然辞职去创业（无铺垫，无原因）

✅ 修正方案：
铺垫：
- 李明对工作不满
- 发现创业机会
- 与家人商量
- 做出决定
```

### 冲突逻辑

**检查要点**：
- 冲突产生是否有原因
- 冲突升级是否有逻辑
- 冲突解决是否合理

**常见问题**：
- 冲突突然爆发（无原因）
- 冲突升级太快
- 冲突解决太容易

**示例检查**：
```
❌ 问题示例：
李明和张伟突然吵架（无原因）
下一集就和好了（无过程）

✅ 修正方案：
铺垫：
- 两人有利益冲突
- 误会加深
- 情绪爆发
- 逐步和解
```

### 反转逻辑

**检查要点**：
- 反转是否有铺垫
- 反转是否合情合理
- 反转是否影响剧情走向

**常见问题**：
- 反转毫无铺垫
- 反转不合逻辑
- 反转影响太大

**示例检查**：
```
❌ 问题示例：
第20集：突然揭露张伟是凶手（无任何铺垫）

✅ 修正方案：
提前埋线索：
- 第5集：张伟在案发现场附近
- 第12集：张伟对案件态度奇怪
- 第18集：张伟有作案动机
```

## 质量要素检查

### 爽点检查

**爽点公式**：装 + 打脸 + 震惊 + 收获

**检查要点**：
- 是否有爽点
- 爽点是否完整
- 爽点是否有力

**常见问题**：
- 爽点不完整
- 爽点太弱
- 爽点频率不当

**示例检查**：
```
✅ 完整爽点示例：
装：李明展示隐藏实力
打脸：反派轻视被打脸
震惊：众人震惊
收获：李明获得资源/地位提升
```

### 爆点检查

**爆点定义**：冲突升级，突破预期

**检查要点**：
- 是否有爆点
- 爆点是否有力
- 爆点位置是否合适

**常见问题**：
- 爆点太弱
- 爆点位置不当
- 爆点过多

**示例检查**：
```
✅ 爆点示例：
李明和反派对峙
突然揭露反派是幕后黑手
（冲突升级，突破预期）
```

### 虐点检查

**虐点定义**：人物遭遇困境，引发同情

**检查要点**：
- 是否有虐点
- 虐点是否合理
- 虐点是否不过分

**常见问题**：
- 虐点太假
- 虐点过分
- 虐点无意义

**示例检查**：
```
✅ 虐点示例：
李明努力工作
老板恶意克扣工资
家人急需用钱
（引发同情，推动剧情）
```

### 钩子检查

**检查要点**：
- 是否有钩子
- 钩子类型是否多样
- 钩子是否与最近3集不同
- 钩子是否有效

**常见问题**：
- 无钩子
- 钩子重复
- 钩子无效
- 钩子不回应

**示例检查**：
```
✅ 有效钩子示例：
第1集：悬念型——他到底是谁？
第3集：回应——揭露他的身份

✅ 钩子多样性检查：
最近3集钩子：悬念型、危机型、转折型
当前钩子：情感型（✅ 不重复）

❌ 钩子重复：
最近3集钩子：悬念型、危机型、转折型
当前钩子：悬念型（❌ 重复）
```

## 节奏检查

### 节奏类型

| 类型 | 特点 | 适用场景 |
|------|------|---------|
| 紧凑 | 动作多，对话少 | 动作戏、高潮 |
| 舒缓 | 描写多，对话多 | 情感戏、铺垫 |
| 适中 | 平衡 | 常规场景 |

### 检查要点

- 整体节奏是否合理
- 是否有节奏变化
- 节奏是否与类型匹配

### 常见问题

- 全程紧张（无喘息）
- 全程平淡（无起伏）
- 节奏突然变化（无铺垫）

## 主题检查

### 检查要点

- 主题是否清晰
- 情节是否服务于主题
- 人物成长是否呼应主题

### 常见问题

- 主题模糊
- 情节脱离主题
- 人物成长无方向
