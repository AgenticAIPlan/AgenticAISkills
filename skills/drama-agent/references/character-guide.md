# 人物设定参考

## 人物小传模板

### 基本信息

| 字段 | 说明 | 示例 |
|------|------|------|
| 姓名 | 全名，符合身份背景 | 李明、张伟、王芳 |
| 年龄 | 具体年龄 | 28岁、30岁 |
| 身份 | 职业、社会地位 | 公司职员、富二代、大学生 |
| 性别 | 男/女 | 男 |

### 性格特点（3-5个关键词）

选择3-5个核心性格词，确保：

1. **具体**：避免太笼统
2. **有特点**：体现人物独特性
3. **可表现**：能通过行为和对话体现

**示例**：

| 角色 | 性格关键词 | 说明 |
|------|-----------|------|
| 李明 | 沉稳、果断、内心脆弱 | 表面沉稳果断，但内心有脆弱一面 |
| 张伟 | 开朗、冲动、讲义气 | 容易冲动，但对朋友很讲义气 |
| 王芳 | 聪明、敏感、自卑 | 很聪明但敏感，内心有自卑 |

### 背景故事（简短）

用2-3句话概括人物背景，包括：

- 出身
- 重大经历
- 影响深远的事件

**示例**：

| 角色 | 背景故事 |
|------|---------|
| 李明 | 出身普通家庭，大学时父亲生病，辍学打工。五年前创业失败，背负巨债。 |
| 张伟 | 富二代，从小衣食无忧。大学时和朋友合伙创业，赚到第一桶金。 |
| 王芳 | 单亲家庭长大，母亲重病。大学时靠奖学金和兼职完成学业。 |

### 核心动机

人物行为的根本驱动力，回答"他/她为什么这么做？"

**常见动机类型**：

| 类型 | 说明 | 示例 |
|------|------|------|
| 生存 | 为了活下去 | 赚钱还债、养家糊口 |
| 复仇 | 为了报复 | 为家人报仇、为自己雪耻 |
| 成功 | 为了证明自己 | 创业成功、获得认可 |
| 爱情 | 为了爱情 | 追求爱人、守护爱情 |
| 守护 | 为了保护重要的人 | 保护家人、朋友 |
| 权力 | 为了获得权力 | 上位掌权、控制他人 |

**示例**：

| 角色 | 核心动机 |
|------|---------|
| 李明 | 偿还债务，给家人更好的生活 |
| 张伟 | 证明自己不是靠家里，有真本事 |
| 王芳 | 守护母亲，实现母亲的期望 |

### 与他人的关系

列出主要人物关系，包括：

| 关系类型 | 说明 | 示例 |
|---------|------|------|
| 亲情 | 父母、兄弟姐妹、子女 | 父亲、姐姐、儿子 |
| 爱情 | 恋人、配偶 | 前妻、女朋友 |
| 友情 | 朋友、同学、同事 | 发小、大学同学、同事 |
| 敌对 | 对手、敌人 | 商业对手、仇人 |
| 复杂 | 爱恨交织 | 前女友（爱恨） |
| 利用 | 互相利用 | 合作伙伴（各有所图） |

**示例**：

| 角色 | 人物 | 关系 | 说明 |
|------|------|------|------|
| 李明 | 张伟 | 朋友 | 大学同学，一起创业 |
| 李明 | 王芳 | 前妻 | 离婚两年，有孩子 |
| 李明 | 陈总 | 对手 | 商业竞争 |
| 张伟 | 王芳 | 暗恋 | 张伟暗恋王芳 |
| 王芳 | 陈总 | 合作 | 工作合作伙伴 |

## 人物类型参考

### 主角类型

| 类型 | 特点 | 适用题材 |
|------|------|---------|
| 成长型 | 从弱到强，经历挫折后成长 | 重生、逆袭、职场 |
| 复仇型 | 被害后重生/觉醒，展开复仇 | 重生复仇、悬疑 |
| 守护型 | 为保护重要的人而战 | 家庭、爱情 |
| 探索型 | 探索未知，寻找真相 | 悬疑、推理 |
| 奋斗型 | 为目标努力奋斗 | 职场、创业 |

### 反派类型

| 类型 | 特点 | 示例 |
|------|------|------|
| 阴谋型 | 精心策划，隐藏很深 | 幕后黑手 |
| 疯狂型 | 情绪化，不可控 | 极端分子 |
| 傲慢型 | 自以为是，看不起人 | 富二代、权贵 |
| 虚伪型 | 表面好人，内心黑暗 | 伪君子 |
| 复杂型 | 有苦衷，立场对立 | 亦正亦邪 |

### 配角类型

| 类型 | 作用 | 示例 |
|------|------|------|
| 助手 | 帮助主角 | 朋友、同事 |
| 阻碍 | 阻碍主角 | 对手、上司 |
| 垫脚石 | 成为主角的踏板 | 傲慢者、看不起主角的人 |
| 工具人 | 推动剧情 | 偶遇的人、服务员 |
| 情感线 | 情感对象 | 恋人、暧昧对象 |

## 人物关系设计

### 关系图谱

```
        李明
       / |  \
      /  |   \
   张伟  王芳  陈总
    |    |
   暗恋  合作
```

### 关系设计原则

1. **清晰**：关系要明确
2. **有层次**：主要关系和次要关系
3. **有张力**：关系要有冲突或变化
4. **推动剧情**：关系要服务于剧情

## 人物成长弧光

### 成长类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 从弱到强 | 能力提升 | 逆袭 |
| 从错到对 | 观念转变 | 觉醒 |
| 从自私到无私 | 价值观变化 | 成长 |
| 从迷茫到坚定 | 目标清晰 | 找到方向 |
| 从封闭到开放 | 心态改变 | 融入社会 |

### 成长节点

1. **起点**：初始状态
2. **触发**：某个事件引发变化
3. **过程**：经历挫折和考验
4. **突破**：关键时刻做出选择
5. **终点**：新的状态

**示例**：

| 阶段 | 李明 |
|------|------|
| 起点 | 普通职员，胆小怕事 |
| 触发 | 发现妻子出轨 |
| 过程 | 离婚、创业失败、还债 |
| 突破 | 决定重新开始，不再逃避 |
| 终点 | 成功创业者，自信坚强 |

## 人物对话参考

### 说话方式差异

不同性格的人说话方式不同：

| 性格 | 说话特点 | 示例 |
|------|---------|------|
| 开朗 | 话多，语气轻快 | "哈哈，太好了！" |
| 沉稳 | 话少，语气平静 | "我知道了。" |
| 冲动 | 语速快，情绪化 | "你怎么能这样！？" |
| 谨慎 | 说话委婉，留余地 | "我觉得可能..." |
| 傲慢 | 语气高傲，看不起人 | "这种事还需要教？" |
| 谦虚 | 语气谦和 | "您说得对。" |

### 对话原则

1. **符合性格**：台词要符合人物性格
2. **推动剧情**：对话要有目的
3. **自然流畅**：像真实对话
4. **有信息量**：传递信息或暗示
5. **避免废话**：不说无关的话

## 常见人物问题

### 问题1：人物太完美

**表现**：主角无缺点，什么都会

**解决**：给人物添加弱点或缺点

### 问题2：人物太扁平

**表现**：人物性格单一，没有层次

**解决**：添加复杂面，如外表开朗内心脆弱

### 问题3：人物行为无动机

**表现**：人物突然做某事，无原因

**解决**：明确人物动机，添加铺垫

### 问题4：人物前后不一致

**表现**：性格或行为突然改变

**解决**：要么添加铺垫，要么调整设定
