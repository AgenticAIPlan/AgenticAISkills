# 剧本格式详细说明

## 基本结构

```
第X集
（集标题，可选）

X-X
[时间]/[内景/外景] [地点]
[人物]

[场景描述]

△ [动作描写]
[人物]：[台词]
OS [心理活动]

【钩子：[钩子类型]——[钩子描述]】
```

## 字段说明

### 集数标题

- 格式：`第X集`
- 可选：添加集标题，如 `第1集 重生归来`
- 作用：帮助读者快速定位

### 场次编号

- 格式：`集数-场次`，如 `1-1`、`1-2`
- 每集从1开始
- 每个场景递增1

### 时间

**日场**：日 / 白天 / 清晨 / 中午 / 下午 / 傍晚
**夜场**：夜 / 深夜 / 凌晨

示例：
```
日
夜
清晨
傍晚
凌晨
```

### 场景类型

- **内景**：室内场景，如房间、办公室、车内
- **外景**：室外场景，如街道、公园、山上
- **内/外景**：内外结合

示例：
```
内景 李明家 客厅
外景 街道
内/外景 车内
```

### 人物

- 列出本场出场的主要人物
- 用顿号或空格分隔
- 按重要性排序

示例：
```
李明、张伟
李明 张伟 王芳
```

### 场景描述

- 描述环境、氛围、初始状态
- 用现在时，简洁明了
- 不超过3行

示例：
```
客厅里一片狼藉，碎玻璃散落一地。
李明坐在沙发上，双手抱头。
窗外传来警笛声，越来越近。
```

### 动作描写（△标记）

- 用 `△` 开头
- 描述人物具体动作
- 暗示情绪状态
- 动词为主，少用形容词

示例：
```
△ 他猛地站起来，狠狠拍了桌子
△ 她颤抖着手，拨通了那个号码
△ 他深吸一口气，推开门
△ 她眼眶发红，强忍着不让眼泪掉下来
```

### 台词

- 格式：`[人物]：[台词]`
- 台词符合人物性格
- 每句台词另起一行
- 不用引号

示例：
```
李明：我决定了，我要离婚。
张伟：你疯了吗？想想孩子！
李明：正因如此，我才不能继续下去。
```

### 心理活动（OS标记）

- 用 `OS` 开头
- 描述人物内心想法
- 可用引号
- 简短有力

示例：
```
OS（他到底想干什么？）
OS（难道他知道了一切？）
OS："对不起，我必须这么做。"
```

### 钩子标注

- 格式：`【钩子：[类型]——[描述]】`
- 场景结尾标注
- 简洁明了

示例：
```
【钩子：悬念型——他手里握着什么？】
【钩子：危机型——时间不多了】
【钩子：转折型——真相出乎意料】
```

## 完整示例

```
第1集 重生归来

1-1
日/内景 李明家 卧室

阳光透过窗帘照进房间，李明猛地睁开眼睛，大口喘息。

△ 他摸了摸自己的脸，确认这不是梦
△ 拿起手机，屏幕显示：2024年6月15日

李明：2024年...我回来了？

OS（那场大火...我竟然重生了？）

△ 他跳下床，冲向阳台
△ 看着楼下车水马龙的街道，泪流满面

李明：这一次，我不会再犯错！

【钩子：悬念型——李明上辈子发生了什么？】

1-2
夜/外景 街道

李明在街上奔跑，神情焦急。

△ 他不停地看手表
△ 拦下一辆出租车

李明：师傅，去市医院！快！

出租车司机：好的，坐稳了！

△ 出租车飞驰而去，在夜色中留下车灯的光芒

【钩子：危机型——李明要去医院做什么？】
```

## 常见格式错误

### 错误1：场次编号混乱
```
❌ 1-1
   1-3
   1-2

✅ 1-1
   1-2
   1-3
```

### 错误2：标记符号不规范
```
❌ （动作）他站起来
❌ 【动作】他站起来
❌ 他站起来。

✅ △ 他站起来
```

### 错误3：台词格式错误
```
❌ 李明："我决定了。"
❌ "李明：我决定了。"

✅ 李明：我决定了。
```

### 错误4：时间地点不清晰
```
❌ 客厅
❌ 日 客厅

✅ 日/内景 李明家 客厅
```

## 格式检查清单

完成剧本后，检查以下项目：

- [ ] 集数编号正确
- [ ] 场次编号连续
- [ ] 时间标注清晰
- [ ] 场景类型明确
- [ ] 地点具体
- [ ] 人物列出完整
- [ ] 场景描述简洁
- [ ] 动作描写用△标记
- [ ] 台词格式正确
- [ ] 心理活动用OS标记
- [ ] 钩子标注规范
- [ ] 没有遗漏标记
- [ ] 标记符号一致
