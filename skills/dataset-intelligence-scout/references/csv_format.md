# CSV格式规范

## 文件夹结构

每次爬取任务创建独立文件夹，精确到秒：

```
工作目录/dataset_scout_YYYY-MM-DD_HH-MM-SS/
├── datasets.csv      # 数据集记录
└── TODO.md           # 任务进度记录
```

示例：`dataset_scout_2026-02-24_14-30-15/`

## 字段说明

| 字段名 | 说明 | 填写规则 |
|--------|------|----------|
| 是否搬运完成 | 标记数据集是否已搬运 | **留空**（后续通过公司接口自动搬运） |
| 数据集名称 | 数据集的名称 | 精确名称，如：xcopa |
| 链接（HF） | Hugging Face 链接 | **留空** |
| 链接（魔搭） | 默认采集渠道的详情页链接 | 完整URL，如：https://www.modelscope.cn/datasets/damotest/xcopa |
| 星河链接 | 星河平台链接 | **留空** |
| 标签 | 数据集标签 | 多个标签用顿号分隔，最多5个，如：智能问答、逻辑推理 |
| 简介 | 数据集简介描述 | 简洁描述，通常1-2句话 |
| 开源协议 | 许可证类型 | 原始值，如：cc-by-4.0, mit, apache-2.0 |
| 大小 | 数据集大小 | 如：64.77 KB, 2.27 MB, 1.17 GB；**未显示则留空** |
| license映射 | license对应的数字ID | 自动映射，见license_mapping.md |
| 抓取渠道 | 数据来源渠道 | 默认填写：魔搭社区；如用户指定其他来源，填写对应渠道 |
| 上传令牌 | 上传时使用的令牌 | 从用户提供的私有令牌池中选择；未提供时填 `<UPLOAD_TOKEN_REQUIRED>` |

## CSV示例行

```csv
,xcopa,,https://www.modelscope.cn/datasets/damotest/xcopa,,智能问答、逻辑推理、知识问答,XCOPA多语言数据集，用于跨语言因果常识推理评估。,cc-by-4.0,64.77 KB,2,魔搭社区,<UPLOAD_TOKEN_REQUIRED>
```

## 上传令牌规则

- 令牌属于私有配置，不要提交到公开仓库。
- 用户提供多个令牌时，每个数据集随机选择一个，避免全部使用同一个令牌。
- 用户未提供令牌时，填写 `<UPLOAD_TOKEN_REQUIRED>`，并在 TODO 中标注待补。

## 大小处理规则

- **≤512GB**：正常登记到CSV
- **>512GB**：跳过，不登记
- **未显示大小**：留空，正常登记

## 大小获取方式

**重要**：大小信息以数据集详情页为准，不在列表页推断。
