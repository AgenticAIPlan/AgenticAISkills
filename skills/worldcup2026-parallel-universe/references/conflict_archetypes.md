# 冲突原型详解

本文档详细说明 WorldCup 2026 Parallel Universe Agent 支持的叙事冲突原型和心理模式。

---

## 冲突原型 (Conflict Archetype)

冲突原型决定了叙事的核心张力和情感走向。

### 1. hero_falls (英雄迟暮)

**适用场景**：
- 传奇球员的最后一战
- 种子球队意外出局
- 老将失误导致败局

**叙事特点**：
- 侧重时间流逝的无奈感
- 强调辉煌与现实的对比
- 结尾带有淡淡的忧伤与释然

**示例三元组**：
```json
{"entity": "C罗", "event_type": "告别演出", "emotion_intensity": 7}
```

**推荐感官锚点**：
- 球场灯光映照下的银发
- 缓慢离场的背影
- 看台上久久不愿离去的球迷

---

### 2. underdog_rises (黑马崛起)

**适用场景**：
- 弱队爆冷晋级
- 新秀球员一战成名
- 赛前不被看好但最终获胜

**叙事特点**：
- 强调逆境中的坚持
- 聚焦"不可能"变为"可能"的瞬间
- 充满希望与激励

**示例三元组**：
```json
{"entity": "日本队", "event_type": "淘汰强队", "emotion_intensity": 8}
```

**推荐感官锚点**：
- 球衣上的汗水与泥土
- 替补席上跃起的身影
- 赛后难以置信的眼神

---

### 3. last_stand (最后一战)

**适用场景**：
- 绝境逆转
- 必须取胜的比赛
- 淘汰赛背水一战

**叙事特点**：
- 极高的紧张感和压迫感
- 强调"没有退路"的决绝
- 高潮部分能量爆发

**示例三元组**：
```json
{"entity": "梅西", "event_type": "绝杀进球", "emotion_intensity": 9}
```

**推荐感官锚点**：
- 最后一秒的倒计时
- 草皮上留下的最后一道痕迹
- 哨声响起前的最后一秒

---

### 4. bitter_betrayal (苦涩背叛)

**适用场景**：
- 点球大战失利
- 关键球员失误
- 近在咫尺却失之交臂

**叙事特点**：
- 强调命运的无情
- 聚焦"差一点"的遗憾
- 带有悲剧色彩但不绝望

**示例三元组**：
```json
{"entity": "巴西队", "event_type": "点球大战失利", "emotion_intensity": 7}
```

**推荐感官锚点**：
- 皮球偏离门柱的几厘米
- 球员跪倒在地的身影
- 看台上骤然凝固的呼吸

---

### 5. silent_redemption (无声救赎)

**适用场景**：
- 之前表现不佳后关键进球
- 从质疑中重新证明自己
- 默默无闻到关键先生

**叙事特点**：
- 内敛而有力的情感
- 强调内心的转变
- 带有哲思和沉淀感

**示例三元组**：
```json
{"entity": "凯恩", "event_type": "关键进球", "emotion_intensity": 8}
```

**推荐感官锚点**：
- 进球后没有狂喜只有平静
- 回望过去的漫长岁月
- 轻轻亲吻队徽的动作

---

## 心理模式 (Psyche Mode)

心理模式决定了叙事的内在情感色彩。

### 1. volcanic (火山爆发式)

**情感特征**：
- 强烈、爆发性
- 情绪外放
- 充满力量感

**适用场景**：绝杀、逆转、夺冠

**叙事语言**：短促有力的句子，大量感叹和动作描写

---

### 2. melancholic (忧郁沉思式)

**情感特征**：
- 内敛、深沉
- 带有淡淡的忧伤
- 思考多于行动

**适用场景**：失利、告别、遗憾

**叙事语言**：长句为主，大量内心独白和意象

---

### 3. defiant (抗争不屈式)

**情感特征**：
- 坚定、不服输
- 带有愤怒和斗志
- 永不放弃

**适用场景**：逆境、被质疑、淘汰出局

**叙事语言**：铿锵有力的宣言，强调决心

---

### 4. numb (麻木疏离式)

**情感特征**：
- 冷漠、抽离
- 仿佛旁观者
- 巨大打击后的失语

**适用场景**：重大失利、意外出局、毁灭性打击

**叙事语言**：极简风格，大量留白和沉默

---

### 5. transcendent (超然升华式)

**情感特征**：
- 平静、通透
- 超越胜负
- 带有哲理和诗意

**适用场景**：传奇告别、完美谢幕、历史性时刻

**叙事语言**：诗化语言，大量比喻和意象，带有时间感
