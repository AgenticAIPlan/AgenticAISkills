# 文生图提示词指南 (Image Prompt Guide)

本文档提供 `ernie-image-turbo` 文生图模型的最佳提示词撰写实践，帮助生成高质量的世界杯主题图片。

---

## 核心原则

### 1. 使用中文提示词

`ernie-image-turbo` 对中文理解能力更强，**所有提示词必须使用中文撰写**。

### 2. 核心事件视觉化

**图片必须能体现出核心事件**。根据三元组中的 `event_type`，选择合适的视觉元素传达事件性质。

**核心事件视觉元素对照表**：

| 核心事件 | 视觉元素建议 | 色彩基调 |
|----------|-------------|----------|
| 绝杀进球 | 球员剪影、球网震颤、金色光芒、胜利姿态、欢腾人群 | 深红/金色 |
| 点球大战失利 | 孤独背影、空旷球门、雨滴、黯淡灯光、沉默的球场 | 灰蓝/银白 |
| 帽子戏法 | 三颗足球、庆祝动作、聚光灯、金色数字"3" | 深紫/金色 |
| 红牌退场 | 红色卡片、离场背影、沉默、灰色调、孤独通道 | 灰色/暗红 |
| 夺冠时刻 | 举起奖杯、金色雨、欢腾人群、国旗飘扬、烟火 | 金色/翠绿 |
| 告别演出 | 挥手告别、夕阳、模糊背影、怀旧色调、散场灯光 | 橙黄/紫色 |
| 淘汰晋级 | 击掌拥抱、团队庆祝、晋级横幅、激情呐喊 | 蓝色/金色 |
| 小组赛出局 | 低头沉默、更衣室背影、失落、空荡球场 | 灰蓝/暗色 |

### 3. 结构化提示词

按照固定结构组织提示词，确保信息完整：

```
[核心事件视觉元素] + [主体描述] + [背景环境] + [色彩基调] + [风格氛围] + [技术参数]
```

### 4. 具体而非抽象

避免模糊描述，使用具体、可感知的词汇：

| ❌ 不推荐 | ✅ 推荐 |
|----------|--------|
| 美丽的背景 | 深蓝色渐变背景，点缀金色光斑 |
| 有氛围感 | 黄昏时分的体育场，灯光穿透薄雾 |
| 简约风格 | 极简构图，大面积留白，中央对齐 |
| 体现比赛 | 球员射门瞬间的剪影，皮球划出金色弧线 |

## 常见问题与解决方案

### 问题1：文字渲染不清晰

**解决方案**：
- 在提示词中明确要求"文字清晰可读"、"高对比度"
- 指定背景与文字的颜色对比（如"深色背景，白色文字"）
- 避免复杂的背景图案干扰文字

### 问题2：风格不符合预期

**解决方案**：
- 增加风格关键词的权重，如"极简风格为主"
- 参考本指南的关键词分类表，使用更精准的词汇
- 添加"避免：[不想要的元素]"明确排除

### 问题3：画面过于复杂

**解决方案**：
- 使用"极简"、"留白"、"干净背景"等关键词
- 明确主体位置（如"画面中央"、"顶部留白"）
- 减少描述性词汇，保留核心元素

### 问题4：情绪表达不准确

**解决方案**：
- 使用本指南的氛围关键词组合
- 结合具体场景描述（如"黄昏"、"雨天"、"聚光灯"）
- 添加感官描述（如"温暖的光线"、"冰冷的色调"）

### 问题5：无法体现核心事件

**解决方案**：
- 参考本指南的"核心事件视觉元素对照表"，选择合适的视觉元素
- 在提示词中明确描述事件相关的视觉符号（如"举起奖杯"、"球网震颤"）
- 添加事件关键词（如"绝杀时刻"、"点球失利"）
- 确保画面主体与事件性质相符（如胜利→庆祝姿态，失利→孤独背影）

## 注意事项

1. **提示词长度**：建议控制在 200-400 字符以内，过长可能导致信息稀释
2. **避免冲突**：不要同时使用相互矛盾的关键词（如"极简"和"丰富细节"）
3. **迭代优化**：首次生成效果不佳时，调整关键词权重或添加具体描述
4. **文化敏感**：避免使用可能引起争议的政治、宗教相关意象
