# 文心大模型产品矩阵速查表

根据项目场景选择最合适的文心模型组合。

## 模型能力一览

| 模型 | 定位 | 适用场景 | 上下文窗口 | 调用方式 |
|------|------|---------|-----------|---------|
| ERNIE 4.5 | 旗舰级 | 复杂推理、长文档理解、代码生成、多模态任务 | 128K | 千帆 API / 私有化 |
| ERNIE 4.5 Turbo | 高性价比 | 大部分生产级场景、内容生成、结构化输出 | 128K | 千帆 API |
| ERNIE Speed | 低延迟 | 实时交互、对话场景、在线客服 | 128K | 千帆 API |
| ERNIE Lite | 轻量级 | 简单任务、高并发、成本敏感场景 | 128K | 千帆 API |
| ERNIE Character | 角色扮演 | 虚拟人、游戏 NPC、数字人对话 | 8K | 千帆 API |

## 场景到模型的选型指南

| 业务场景 | 推荐模型 | 选型理由 |
|---------|---------|---------|
| 智能客服 / 在线问答 | ERNIE Speed | 低延迟满足实时对话需求 |
| 内容生成 / 营销文案 | ERNIE 4.5 Turbo | 生成质量高、性价比优 |
| 文档理解 / 合同分析 | ERNIE 4.5 | 长文档理解能力强 |
| 知识检索增强（RAG） | ERNIE 4.5 Turbo + 百度向量数据库 VDB | 检索与生成解耦、成本可控 |
| 代码生成 / 技术辅助 | ERNIE 4.5 | 代码理解与生成能力最强 |
| 数据分析 / 报告生成 | ERNIE 4.5 Turbo | 结构化输出能力好 |
| 高并发简单分类 / 打标签 | ERNIE Lite | 成本最低、吞吐量大 |
| 虚拟角色 / 数字人 | ERNIE Character | 专为角色扮演优化 |
| 复杂多步骤 Agent 工作流 | ERNIE 4.5 + AppBuilder | 工具调用与编排能力强 |

## 百度生态配套服务

| 服务 | 用途 | 适用场景 |
|------|------|---------|
| 千帆 ModelBuilder | 模型精调、评估、部署 | 需要领域适配或私有数据微调 |
| 千帆 AppBuilder | 应用编排、工具调用、Agent 构建 | 需要多工具协同的 Agent 场景 |
| 百度向量数据库 VDB | 向量存储与检索 | RAG 知识库、语义搜索 |
| 百度智能云 BOS | 训练数据存储 | 大规模数据集管理 |
